## 1. 产品概述

像素风格的星际矿场经营模拟游戏，玩家扮演外星采矿公司经理，在随机生成的星球表面部署采矿机器人采集矿物并换取积分。
- 核心玩法：策略布局、资源管理、升级系统
- 目标用户：休闲策略游戏爱好者
- 市场价值：轻量级网页游戏，易上手、可玩性高

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 完整游戏体验 |

### 2.2 功能模块

1. **游戏主界面**：600x600像素星球地表画布、机器人采矿动画、矿物采集进度显示
2. **机器人系统**：机器人部署、移动寻路、采集状态机、进度条显示
3. **资源系统**：四种矿物（铁、铜、水晶、暗物质）、积分系统
4. **升级商店**：移动速度升级、采集容量升级、购买新机器人

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-----------|----------|
| 游戏主界面 | 星球画布 | 600x600俯视像素地图，点击空地部署机器人 |
| 游戏主界面 | 机器人动画 | 像素移动动画、采集进度条、状态切换 |
| 游戏主界面 | 暗物质特效 | 采集暗物质时全屏紫色光晕闪烁 |
| 右侧面板 | 积分显示 | 当前积分余额实时更新 |
| 右侧面板 | 矿物库存 | 铁/铜/水晶/暗物质四种库存显示 |
| 右侧面板 | 升级商店 | 三项升级按钮，点击有闪烁动画 |

## 3. 核心流程

玩家进入游戏 → 查看随机生成的星球地图 → 初始获得1台免费机器人 → 点击空地部署机器人 → 机器人自动寻路到最近矿脉 → 开始采集 → 采集进度条显示 → 满载自动返回基地 → 矿物入库换取积分 → 使用积分升级或购买新机器人 → 循环采集更多稀有矿物

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深色太空主题，背景#0B0B1E，卡片#1A1A3E，边框#3A3A6E
- 按钮风格：霓虹蓝#00D4FF，圆角像素风
- 像素风格字体，等宽字体增强复古感
- 矿物颜色：铁#808080、铜#D2691E、水晶#8A2BE2、暗物质#1A1A1A带光晕
- 暗物质采集特效：全屏边缘半透明紫色光晕渐变

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|-----------|-----------|---------|
| 游戏主界面 | 星球画布 | 600x600画布、像素瓷砖、矿脉斑点、机器人精灵、进度条 |
| 游戏主界面 | 标题栏 | "星辰矿场 v1.0" 像素字体标题 |
| 右侧面板 | 积分区域 | 大号积分数字、霓虹蓝高亮 |
| 右侧面板 | 库存区域 | 四种矿物图标+数字 |
| 右侧面板 | 升级商店 | 三个升级按钮、价格标签、点击闪白动画 |

### 4.3 响应式设计

- 桌面端（≥800px以上）：双栏布局，左侧画布，右侧面板
- 移动端（<800px以下）：上下堆叠布局，画布在上，面板在下
- 画布保持600x600固定尺寸
- 触控优化支持触摸事件

### 4.4 动画与帧率

- 所有动画帧率稳定30FPS以上
- requestAnimationFrame驱动
- 按钮点击0.2秒闪白动画
- 暗物质采集0.5秒紫色光晕
- 机器人像素移动动画平滑过渡
