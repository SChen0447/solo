## 1. 产品概述

「书痕·光签集」是一款为小型独立书店打造的在线互动书评平台，让顾客能以沉浸式翻页动画浏览书籍，通过拖拽情感标签的方式为书籍添加个性化书评，并以彩色发光书签墙的形式展示所有读者的情感印记。

- **目标用户**：独立书店的顾客、文学爱好者、喜欢分享阅读感受的读者
- **核心价值**：将实体书店的翻阅体验数字化，用情感标签代替传统文字评论，创造独特的视觉化书评展示方式

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览书架、查看书籍详情、拖拽添加情感标签、查看书签墙、按标签筛选 |

### 2.2 功能模块
1. **首页**：虚拟书架展示、发光书签墙、情感标签筛选
2. **书籍详情**：3D翻页动画、书籍简介展示、情感标签拖拽添加

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 虚拟书架 | 展示所有书籍封面卡片，支持悬停动画，点击进入详情 |
| 首页 | 书签墙 | 水平滚动展示所有书评书签，彩色发光效果，悬停放大旋转 |
| 首页 | 标签筛选栏 | 6个彩色圆点按钮，点击筛选对应情感标签的书签 |
| 书籍详情页 | 翻页动画区 | 3D翻转效果展示封面和书籍简介、作者、出版年份 |
| 书籍详情页 | 侧边标签栏 | 6种预设情感标签，可拖拽至书页区域 |
| 书籍详情页 | 书页区域 | 接收拖拽的标签，放下时产生波纹动画，自动提交书评 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深木色渐变背景（#1a0f0a → #0d0806），书架区域（#2c1810 → #3e2316）
- **强调色**：金色光晕 rgba(255,215,0,0.3)，6种情感标签色（温暖#ff6b6b、忧伤#74b9ff、智慧#48dbfb、浪漫#fd79a8、神秘#a29bfe、治愈#00b894）
- **卡片风格**：圆角16px，毛玻璃效果 backdrop-filter:blur(12px)，背景 rgba(255,255,255,0.06)，1px发光边框
- **字体**：中文衬线体用于标题，无衬线体用于正文
- **动画**：所有交互使用 transform 和 opacity 实现GPU加速，过渡时长0.3-0.6s

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 虚拟书架 | 深色木纹背景、160x220px发光封面卡片、悬停抬起阴影、右下角情感标签光点 |
| 首页 | 书签墙 | 高300px水平滚动区、120x160px渐变发光书签、40x40px圆形缩略图、悬停放大1.1倍旋转2度 |
| 首页 | 标签筛选栏 | 6个24px直径彩色圆点、选中放大1.2倍带白色光晕 |
| 书籍详情页 | 侧边标签栏 | 固定200px宽、毛玻璃背景、高40px圆角标签、透明度0.4 |
| 书籍详情页 | 翻页区域 | CSS 3D Y轴旋转180度、0.6s动画、边缘发光拖尾、白色14px文字行距1.8 |

### 4.3 响应式设计
- **桌面端（≥768px）**：左侧200px固定侧边栏，右侧自适应内容区，书签墙水平滚动
- **移动端（<768px）**：侧边栏折叠为顶部汉堡菜单，书签墙改为垂直滚动，卡片尺寸自适应缩小

### 4.4 性能优化
- 所有CSS动画使用 transform 和 opacity 属性确保GPU加速
- 翻页和筛选响应时间 ≤ 300ms
- 书签墙滚动保持60fps
- 避免使用JavaScript动画，优先CSS transition/animation
