## 1. 产品概述

虚拟香氛蜡烛工坊与气味时空旅行 — 一款让用户化身气味设计师的全栈Web应用。用户可在浏览器中调配不同香调（前调、中调、基底调），按比例混合注入虚拟蜡烛，点燃后观察气味分子扩散的粒子动画，体验动态时空气味旅行。

- 目标用户：对香氛调配感兴趣的大众用户、创意爱好者
- 核心价值：将抽象的嗅觉体验转化为可视化的交互艺术，提供沉浸式气味设计体验

## 2. 核心功能

### 2.1 功能模块

1. **工坊主页**：木质手作工坊界面，三根蜡烛展示与选择，配方卡片网格
2. **香调配比工作台**：12格扇形配比盘，旋钮/输入控制，LED灯珠反馈
3. **蜡烛燃烧体验**：灯芯点燃、蜡面融化、颜色过渡、粒子扩散动画
4. **配方管理**：localStorage保存/加载，卡片展示与恢复

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工坊主页 | 木质桌面场景 | 从#2c1810渐变到#4a2c2a再渐变到#8b5a2b的背景，CSS纹理桌面，三根蜡烛展示 |
| 工坊主页 | 蜡烛选择 | 点击蜡烛切换到对应配方工作台，蜡烛直径80px高120px，蜂蜜黄/玫瑰粉/森绿三色 |
| 工坊主页 | 配方卡片网格 | 100px×80px圆角卡片，背景色为蜡烛颜色，左下角燃烧进度条，点击恢复状态 |
| 香调配比工作台 | 12格扇形配比盘 | 半径140px圆形盘，12扇形格代表12种香调精油，0-100%浓度渐变 |
| 香调配比工作台 | 旋钮/输入控制 | 拖动旋钮或输入数字0-100调整百分比，实时计算总浓度≤100% |
| 香调配比工作台 | LED灯珠指示 | 配比盘外圈4px圆点灯珠，浓度>60%时点亮金色#ffd700并脉动（亮度1→0.6，周期1s） |
| 蜡烛燃烧体验 | 灯芯点燃 | 点击蜡烛顶部灯芯开始燃烧，触发融化与粒子动画 |
| 蜡烛燃烧体验 | 蜡面融化 | CSS clip-path模拟融化边缘，融面每30秒下降5px |
| 蜡烛燃烧体验 | 颜色过渡 | 燃烧>2分钟后蜡色向焦棕#5a3e2b过渡，每20s变化1/60 |
| 蜡烛燃烧体验 | 粒子扩散 | Canvas逐帧渲染，各香调对应不同颜色/形状粒子，发射30-50个/秒，扩散半径300px→200px，透明度4秒衰退 |
| 蜡烛燃烧体验 | 香气减弱 | 燃烧后粒子扩散半径从300px缩小到200px，模拟挥发减弱 |

## 3. 核心流程

用户进入工坊主页 → 看到三根蜡烛 → 点击选择一根蜡烛 → 右侧配比盘出现 → 拖动旋钮/输入数值调配香调比例 → 确认总浓度≤100% → 点击蜡烛灯芯点燃 → 观察蜡面融化与粒子扩散 → 燃烧2分钟后颜色渐变、扩散范围缩小 → 保存配方到localStorage → 返回主页查看配方卡片 → 点击卡片恢复上次状态

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：木质暖棕色系（#2c1810 → #4a2c2a → #8b5a2b）
- 辅助色：宣纸色文字#f5e6c8，深棕按钮#3e2723，金色描边#d4af37
- 按钮：深棕色底+金色描边，hover时光泽变化
- 字体：衬线体为主，营造手作工坊氛围
- 布局：桌面中央为操作区，右侧配比盘，下方配方卡片
- 动画：CSS transition 0.3s ease-out 统一交互过渡

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工坊主页 | 木质桌面场景 | 渐变背景+CSS噪点纹理桌面，温暖木质氛围 |
| 工坊主页 | 蜡烛展示 | 三个圆柱体蜡烛，凹凸质感+分层浇铸纹理，不同蜡色 |
| 工坊主页 | 配方卡片 | 100×80px卡片网格，8px圆角，蜡烛颜色背景，燃烧进度条 |
| 香调配比工作台 | 配比盘 | 140px半径圆形盘，12扇形格，浓度渐变色，外圈LED灯珠 |
| 香调配比工作台 | 控制区 | 旋钮拖动+数字输入，实时浓度指示 |
| 蜡烛燃烧体验 | 粒子动画 | Canvas全屏覆盖，多色多形态粒子，FPS≥40 |

### 4.3 响应式

- 桌面优先设计
- 宽屏(>1024px)：蜡烛并排三列，配比盘140px
- 平板(768-1024px)：两列布局，配比盘半径100px
- 手机(<768px)：单列堆叠，蜡烛与配比盘上下排列
- 触屏优化：拖拽旋钮和点击灯芯均支持touch事件
