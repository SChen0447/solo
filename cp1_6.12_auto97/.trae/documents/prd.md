## 1. 产品概述

涂鸦工坊是一个面向街头艺术家的数字涂鸦作品展示与粉丝互动平台，解决传统社交媒体中涂鸦作品缺乏沉浸式展示和即时反馈的问题。通过模拟真实墙面的涂鸦卡片布局、Canvas像素级涂鸦效果渲染以及实时点赞评论互动，为街头艺术家打造专属的数字创作空间。

- 目标用户：街头艺术家、涂鸦创作者、街头文化爱好者
- 核心价值：沉浸式涂鸦展示体验 + 即时粉丝互动反馈

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 街头艺术家 | 默认登录 | 管理作品集、上传新作、查看粉丝、互动管理 |

### 2.2 功能模块

1. **涂鸦墙（作品墙）**：动态涂鸦墙主视图、卡片展开动画、作品详情展示
2. **作品详情**：Canvas涂鸦效果渲染、作品信息编辑、点赞评论互动
3. **上传新作**：文件拖拽上传、进度条动画、跳转编辑
4. **我的粉丝**：粉丝列表展示、移除粉丝操作
5. **导航栏**：页面切换、下划线跟随动画

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 涂鸦墙 | 涂鸦卡片网格 | 20张随机分布卡片，随机旋转和Y轴偏移，彩色径向渐变背景，点击展开至详情 |
| 涂鸦墙 | 卡片展开动画 | 从角落放大至视口70%宽度，背景变暗遮罩，0.4s ease-out |
| 作品详情 | 涂鸦效果渲染 | 图片转黑白+30%概率随机彩点喷溅，Canvas像素操作 |
| 作品详情 | 作品信息卡 | 标题编辑、日期自动记录、标签增删、点赞动画、评论气泡 |
| 上传新作 | 文件上传区域 | 虚线边框拖拽区，上传后跳转详情+进度条动画 |
| 我的粉丝 | 粉丝列表 | 头像+昵称+关注时间，移除按钮hover变红 |
| 导航栏 | 顶部固定导航 | Logo+4导航按钮，渐变下划线跟随动画 |

## 3. 核心流程

### 3.1 浏览作品流程
用户进入涂鸦墙 → 浏览随机分布的涂鸦卡片 → 点击感兴趣的卡片 → 卡片展开动画播放 → 进入作品详情 → 查看涂鸦效果/点赞/评论 → 点击遮罩或关闭返回涂鸦墙

### 3.2 上传作品流程
用户点击"上传新作" → 进入上传页面 → 拖拽或选择图片 → 进度条动画播放 → 自动跳转作品详情编辑 → 填写标题/标签 → 完成发布

### 3.3 粉丝管理流程
用户点击"我的粉丝" → 查看粉丝列表 → 可移除粉丝

## 4. 用户界面设计

### 4.1 设计风格

- **风格定位**：深色街头艺术风格，粗犷与精致并存
- **主色调**：黑色(#0A0A0A)与深灰(#1E1E1E)为主
- **点缀色**：#FF3366亮粉红、#33CCFF亮蓝、#FFD700金色
- **街头色板**：#FF3366、#33CCFF、#FFD700、#AA66CC、#66FF66
- **按钮风格**：圆角半透明，hover变不透明
- **字体**：标题加粗白色，正文灰色
- **布局风格**：卡片网格+固定顶部导航
- **动画统一**：0.4s duration，ease-in-out缓动

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 涂鸦墙 | 背景层 | 深灰到黑色径向渐变(#1E1E1E→#0A0A0A) |
| 涂鸦墙 | 卡片层 | 正方形卡片120-180px，随机旋转(-5°~5°)，Y轴偏移，彩色径向渐变+涂鸦预览图标 |
| 涂鸦墙 | 遮罩层 | 展开时背景变暗 |
| 作品详情 | 左侧 | Canvas渲染涂鸦效果，图片自适应 |
| 作品详情 | 右侧 | 半透明信息卡：标题输入框、日期、标签胶囊(半透明白色圆角)、心形点赞+动画、评论气泡(20px圆角半透明白) |
| 上传页面 | 上传区 | 虚线边框500×300px，圆角16px，半透明灰底 |
| 上传页面 | 进度条 | #33CCFF从0%→100%，2s动画 |
| 粉丝页面 | 粉丝行 | 圆形头像50px(随机色板色)、昵称、X天前、红色移除按钮 |
| 导航栏 | 整体 | 固定顶部60px，rgba(0,0,0,0.6)+blur(8px) |
| 导航栏 | Logo | 喷漆罐图标+"涂鸦工坊"加粗白色 |
| 导航栏 | 下划线 | #FF3366→#33CCFF渐变，跟随切换0.3s动画 |

### 4.3 响应式设计

- 桌面优先（≥768px）：涂鸦墙4列、详情左右分栏、导航完整文字
- 移动端（<768px）：涂鸦墙2列、详情上下堆叠、导航图标+文字缩写

### 4.4 性能要求

- 点击卡片到展开动画开始：≤100ms
- Canvas涂鸦效果生成：≤500ms
