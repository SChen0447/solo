## 1. 产品概述

奇幻生物孵化与成长模拟沙盒——一款面向宠物爱好者和幻想题材玩家的网页沉浸式培育体验应用。玩家可以从魔法蛋池中选择蛋进行孵化，培育幼体生物，投喂食物提升属性，将不同生物融合创造全新混合种，并在自然栖息场景中观察生物自由走动与社交互动。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册，直接使用 | 选择蛋、孵化、投喂、融合、查看属性 |

### 2.2 功能模块

1. **主页面**：蛋池选择区、孵化巢、栖息场景、生物属性面板、融合区域

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 蛋池选择区 | 展示5种魔法蛋（龙、独角兽、凤凰、狮鹫、石像鬼），点击后蛋出现在孵化巢中 |
| 主页面 | 孵化巢 | 蛋展示区域，动态流光粒子背景，蛋裂纹动画，破壳弹出幼体，出生光环扩散效果 |
| 主页面 | 投喂系统 | 5种食物（魔力浆果、月光露、火焰晶、冰霜草、暗影菇），每种提升特定属性+3~7，0.3%变异概率 |
| 主页面 | 融合系统 | 拖拽两只不同生物进行融合，漩涡粒子效果1.5秒，产生混合种，颜色插值混合，0.2%远古形态概率 |
| 主页面 | 栖息场景 | 草地背景，云朵飘动，星星闪烁，生物A*寻路自由走动，社交互动（鸣叫、爱心/音符动画） |
| 主页面 | 属性面板 | 毛玻璃效果，生物头像、6项属性进度条（红黄绿变色）、心情表情图标 |

## 3. 核心流程

1. **孵化流程**：玩家从蛋池选择蛋 → 蛋出现在孵化巢 → 流光粒子背景 → 蛋裂纹动画0.8秒 → 破壳弹出幼体0.3秒ease-out → 光环扩散0.5秒 → 幼体生物进入场景
2. **投喂流程**：选择生物 → 点击食物 → 属性+3~7 → 属性增长闪烁动画0.5秒 → +数字飘字效果 → 0.3%概率触发变异
3. **融合流程**：拖拽生物A到生物B → 漩涡粒子1.5秒 → 产生混合种（名字组合、颜色插值）→ 属性取平均值 → 0.2%概率远古形态
4. **互动流程**：生物在场景中A*寻路走动 → 靠近时互相鸣叫 → 爱心/音符符号1.2秒 → 点击生物查看属性面板

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深紫色（#2b1b3d）背景，柔和金色（#d4a373）和浅蓝（#a0c4ff）装饰高亮
- 按钮样式：圆角矩形，金色光泽box-shadow发光阴影
- 字体：奇幻绘本风格衬线字体，标题使用装饰性字体，正文使用清晰易读字体
- 布局：flexbox和grid混合布局，手机端自动转竖向
- 交互效果：悬停放大1.05倍，点击脉冲（scale(0.95)后恢复），孵化巢和融合区呼吸光效（opacity 0.8-1循环，周期2s）

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 蛋池选择区 | 深紫卡片，金色描边蛋图标，悬停发光放大，5个蛋水平排列（手机竖排） |
| 主页面 | 孵化巢 | 居中区域，流光粒子背景，呼吸光效边框，蛋裂纹动画SVG叠加 |
| 主页面 | 栖息场景 | 全宽Canvas画布，草地渐变底色，云朵和星星层，生物精灵走动带脚印 |
| 主页面 | 属性面板 | 半透明毛玻璃（backdrop-filter: blur(10px)），圆角，属性条红黄绿渐变，心情emoji |
| 主页面 | 食物栏 | 底部水平食物卡片，图标+名称，点击脉冲动画 |
| 主页面 | 融合区 | 双槽位拖拽区，呼吸光效，漩涡粒子效果 |

### 4.3 响应式设计

- 桌面优先设计，使用flexbox和grid布局
- 手机端（<768px）：布局从横向转为竖向，场景画布缩小，面板全屏弹出
- 触摸优化：拖拽融合支持触摸事件，按钮点击区域增大

### 4.4 动画与性能

- 孵化巢和融合区域呼吸光效：opacity 0.8-1循环，周期2s
- 生物行走脚印：半透明，随移动逐渐消失
- 粒子效果：速度0.5-2px/s，暖色系随机颜色
- 融合粒子：每帧≤300个粒子
- 行走动画帧率：≥55FPS
- 同时≤5只生物不卡顿
