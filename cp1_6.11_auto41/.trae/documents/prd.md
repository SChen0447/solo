## 1. 产品概述
时空旅行灵感探索工具，用户在世界地图上点击任意地点，即可探索该地点的"平行时空"——历史古迹复原、当前实时环境数据、未来气候模拟。
- 核心价值：以沉浸式时空视角激发旅行灵感，融合历史人文与科学展望
- 目标用户：旅行爱好者、历史文化爱好者、环境关注者

## 2. 核心功能

### 2.1 功能模块
1. **首页地图**：交互式世界地图、时空锚点标记、悬停预览卡片、点击飞入动画
2. **时空详情窗**：三标签页切换（过去/现在/未来）、动态可视化图表、时间轴交互

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 交互式地图 | Canvas渲染世界地图，支持拖拽缩放，帧率≥30fps |
| 首页 | 时空锚点 | 随机散布圆形标记，颜色按纬度渐变（极地蓝→赤道橙红） |
| 首页 | 悬停交互 | 鼠标悬停标记放大，显示地名和简介预览卡片 |
| 首页 | 飞入动画 | 点击标记后地图平滑飞入目标城市 |
| 详情窗 | 过去标签 | 古建线稿复原图，褪色纸张纹理背景，纵向时间轴滑块控制渐显/渐隐 |
| 详情窗 | 现在标签 | 实时天气与空气质量数据，动态环形图展示，0.5秒弹性动画更新 |
| 详情窗 | 未来标签 | 气候模拟海报，温度曲线随时间轴波动，背景色冷蓝→暖橙平滑过渡 |

## 3. 核心流程
用户打开首页 → 加载交互式世界地图与随机时空锚点 → 鼠标悬停锚点查看预览 → 点击感兴趣的锚点 → 地图平滑飞入该城市 → 弹出时空详情窗 → 切换三个标签页探索不同时空维度 → 拖动时间轴交互查看动态变化

## 4. 用户界面设计

### 4.1 设计风格
- **配色方案**：极简档案风格——米白底色(#F5F1E8)、深灰文字(#2C2C2C)、低饱和度蓝绿色系边框(#5A8A8A)
- **卡片样式**：细边框、轻微阴影、12px圆角
- **字体**：标题使用"Noto Serif SC"衬线字体，正文使用"Noto Sans SC"无衬线字体，营造档案文献感
- **动效**：0.3秒fade-in + 轻微上移的卡片切换动画

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 地图区域 | 全屏Canvas地图、圆形锚点标记、渐变色彩、悬停放大动画 |
| 首页 | 预览卡片 | 半透明米白背景、细蓝绿边框、地名+一句话简介 |
| 详情窗 | 标签栏 | 三个等宽标签、下划线指示当前激活态 |
| 详情窗 | 过去页 | 古建线稿SVG、纸张纹理背景、纵向时间滑块、年份渐显效果 |
| 详情窗 | 现在页 | 动态环形图（天气/空气质量）、数值标签、弹性动画 |
| 详情窗 | 未来页 | 温度曲线图、渐变背景过渡、时间轴推进动画 |

### 4.3 响应式
- **桌面端**：详情窗从右侧滑入，占屏幕45%宽度，高度自适应
- **移动端**（<768px）：详情窗从底部滑入占满屏幕宽度，标签页变为可横向滑动的Tab条，内容竖向堆叠
- **触摸优化**：锚点触摸热区≥44x44px，支持双击缩放地图

## 5. 性能要求
- 地图拖拽/缩放帧率≥30fps
- 详情窗内所有动画流畅不卡顿
- 首次加载所有SVG/Canvas元素≤2秒完成渲染
- 时间轴滑块实时响应无延迟
