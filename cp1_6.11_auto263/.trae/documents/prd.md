## 1. 产品概述

虚拟街头涂鸦墙与动态颜料在线协作平台——让多位用户在同一面虚拟砖墙上实时涂鸦，模拟真实喷罐的飞溅、滴落效果，颜料随时间风干变色，重复覆盖产生立体堆积纹理。

- 目标用户：街头艺术爱好者、创意工作者、在线协作团队
- 核心价值：以高度仿真的喷罐物理效果和实时多人协作，打造沉浸式数字涂鸦体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 涂鸦艺术家 | 自动分配昵称 | 在砖墙上喷绘、选择颜色/喷嘴、查看历史记录 |

### 2.2 功能模块
1. **涂鸦墙主界面**：全屏砖墙Canvas、右侧调色工具栏、左侧操作记录面板、底部颜色历史
2. **喷罐涂鸦引擎**：飞溅散点绘制、滴落效果、颜色风干老化、颜料堆积凸起

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 涂鸦墙主界面 | 砖墙Canvas | 从旧砖红(#8b4513)到灰泥色(#c0c0c0)纹理拼接，砖块宽80px高40px，浅浮雕阴影效果 |
| 涂鸦墙主界面 | 调色工具栏 | 10种颜色圆形色块(直径35px)，点击外发光选中，喷嘴大小/压力调节滑块 |
| 涂鸦墙主界面 | 喷罐绘制 | 鼠标拖动喷绘，随机散布小圆点(3-8px)，密度与速度反比，末端3-5个滴落线条(10-30px) |
| 涂鸦墙主界面 | 颜色老化 | 每30秒RGB向灰色偏移10%，2分钟后饱和度降为0，记录初始时间戳 |
| 涂鸦墙主界面 | 颜料堆积 | 直径50px内重复喷绘超3次，Canvas阴影偏移+scale凸起(3-5px)，保留纹理高光 |
| 涂鸦墙主界面 | 多人协作 | WebSocket实时广播涂鸦数据，显示其他用户光标和昵称，在线人数统计 |
| 涂鸦墙主界面 | 操作记录面板 | 左侧浅灰半透明面板，最近5条涂鸦操作（昵称+颜色名） |
| 涂鸦墙主界面 | 颜色历史 | 底部显示最近使用的3种颜色缩略图 |

## 3. 核心流程

用户进入页面 → 看到全屏砖墙和工具栏 → 选择颜色和喷嘴参数 → 在墙上拖动鼠标喷绘 → 喷罐效果实时渲染（飞溅+滴落） → 颜色随时间风干老化 → 重复区域颜料堆积凸起 → 操作通过WebSocket广播 → 其他用户实时看到涂鸦

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暗黑工业风背景(#1a1a1a)衬托涂鸦鲜艳色
- 辅助色：砖红(#8b4513)、灰泥(#c0c0c0)、浅灰面板(#rgba(200,200,200,0.15))
- 按钮风格：圆形色块，选中时外发光
- 字体：Rajdhani（工业感标题）+ Source Sans 3（UI文本）
- 布局：涂鸦墙居中70%，右侧固定工具栏，左侧半透明操作面板

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 涂鸦墙主界面 | 砖墙区域 | Canvas全屏，砖块纹理+浅浮雕阴影，深色背景 |
| 涂鸦墙主界面 | 调色工具栏 | 右侧固定面板，10个圆形色块垂直排列，喷嘴/压力滑块 |
| 涂鸦墙主界面 | 操作记录 | 左侧浅灰半透明面板，5条记录列表 |
| 涂鸦墙主界面 | 颜色历史 | 底部3个颜色缩略图 |
| 涂鸦墙主界面 | 在线状态 | 顶部显示在线人数和用户光标 |

### 4.3 响应式
- 桌面优先设计
- 屏幕宽度 < 800px 时工具栏折叠为顶部横向布局
- 触摸屏优化：支持触摸喷绘操作

### 4.4 动效
- 所有过渡动画使用 ease-out 曲线
- 色块选中时外发光动画
- 颜色老化渐变过渡
- 涂鸦操作延迟 ≤ 100ms
