## 1. 产品概述
在线协作白板应用，支持团队成员实时绘制图形、添加便签和标注，并具备历史版本回溯能力。
- 核心目标：提供高效的远程协作工具，支持多人实时同步绘图操作
- 目标用户：远程办公团队、设计团队、教育工作者
- 市场价值：填补轻量级实时协作白板工具的市场空白，支持操作转换和版本管理

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 一般用户 | 输入昵称加入房间 | 可绘制图形、同步操作、查看历史版本 |

### 2.2 功能模块
1. **房间入口页**：昵称输入、房间号生成/输入、加入按钮
2. **白板主界面**：Canvas画布、左侧工具栏、顶部状态栏、历史回放滑块
3. **绘图系统**：画笔、矩形、圆形、文本、橡皮擦、选择工具
4. **实时同步系统**：Socket通信、操作转换、冲突解决
5. **版本管理系统**：自动快照、历史回放、版本切换

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 房间入口页 | 登录表单 | 输入昵称，生成/输入4位房间号，点击加入 |
| 白板主界面 | 顶部导航栏 | 显示房间号、在线人数、历史回放滑块 |
| 白板主界面 | 左侧工具栏 | 画笔、矩形、圆形、文本、橡皮擦、选择工具、颜色选择 |
| 白板主界面 | 画布区域 | 支持绘制、选择、缩放、平移、元素操作 |
| 白板主界面 | 底部状态栏 | 显示当前版本号 |

## 3. 核心流程
用户输入昵称和房间号后加入房间，进入白板界面。通过左侧工具栏选择绘图工具进行创作，操作通过Socket实时同步给其他在线用户。用户可通过顶部滑块回放历史版本，每30秒或操作后自动保存快照。

```mermaid
flowchart TD
    "A[进入页面]" --> "B[输入昵称/房间号]"
    "B" --> "C[点击加入]"
    "C" --> "D[连接Socket服务]"
    "D" --> "E[进入白板界面]"
    "E" --> "F[选择绘图工具]"
    "F" --> "G[绘制操作]"
    "G" --> "H[本地渲染]"
    "H" --> "I[发送操作到服务器]"
    "I" --> "J[服务器广播给其他用户]"
    "J" --> "K[其他用户渲染]"
    "E" --> "L[拖动历史滑块]"
    "L" --> "M[请求版本快照]"
    "M" --> "N[恢复到指定版本]"
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色：#4A90D9（蓝色）
- 辅色：#F0F4F8（浅灰蓝）
- 白板背景：#F5F5F5
- 工具栏背景：#FFFFFFCC（半透明白色）
- 字体：现代无衬线字体，标题稍大
- 布局：桌面端左侧工具栏，移动端顶部汉堡菜单
- 图标：SVG内联图标，选中时高亮

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 房间入口页 | 登录卡片 | 居中卡片、输入框、按钮、阴影效果 |
| 白板主界面 | 顶部导航 | 深色背景(#1E293B)、房间号、在线人数、滑块 |
| 白板主界面 | 左侧工具栏 | 半透明背景、竖排工具按钮、颜色选择器 |
| 白板主界面 | 画布区域 | 灰色背景、Canvas元素、光标变化 |
| 白板主界面 | 底部状态栏 | 版本号显示、居中 |

### 4.3 响应式设计
- 桌面端（≥768px）：左侧固定工具栏（宽50px）
- 移动端（<768px）：顶部汉堡菜单，工具栏展开为下拉面板
- 触摸操作优化：支持触摸绘制和手势缩放

### 4.4 动画效果
- 元素创建：淡入效果 0.2s（opacity 0→1）
- 元素删除：缩放消失 0.15s（scale 1→0，opacity 1→0）
- 工具按钮选中：背景色过渡 0.2s
- 画布缩放：平滑过渡 0.3s
