## 1. 产品概述

面向学生和科研爱好者的交互式3D分子结构与化学键合可视化应用，解决在没有专业软件的情况下直观观察并操作分子模型、理解化学键形成与断裂过程的问题。通过浏览器即可实现360度旋转、分子切换、键合模拟等核心功能，降低学习门槛。

## 2. 核心功能

### 2.1 功能模块

1. **3D分子渲染页面**：球棍模型展示、鼠标旋转/缩放、化学键悬停信息
2. **分子切换系统**：侧边栏选择预设分子，平滑动画过渡，粒子特效
3. **键合模拟模式**：点击原子创建/断裂化学键，实时更新分子信息
4. **信息展示面板**：分子式、原子数量、键数、价态信息表

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 3D渲染区域 | 加载球棍模型，原子不同颜色球体+化学键半透明圆柱，鼠标拖拽360度旋转（惯性0.5），滚轮缩放0.2-5倍 |
| 主页面 | 化学键悬停 | 鼠标悬停键时颜色变淡蓝#66b3ff，显示键类型和键能标签 |
| 主页面 | 分子切换 | 侧边栏选择H2O/CH4/CO2/NH3，原子飞向新位置1.5s ease-in-out，新键形成青色粒子扩散，旧键断裂红色粒子消散 |
| 主页面 | 键合模拟 | 点击原子高亮亮黄色光圈，点击第二原子添加新键（0.5s延展动画）或断裂已有键（变细+红色+消失+弹开） |
| 主页面 | 环境粒子 | 半透明浅青蓝小点随机分布100单位半径球体内，速度0.02/帧 |
| 主页面 | 左侧面板 | 140px窄栏，分子选择按钮+模式切换，渐变背景，hover上移2px 0.2s，点击涟漪0.3s |
| 主页面 | 右侧面板 | 280px宽栏，分子式、原子数量、键数、价态表格（交替半透明白/冷桃色背景，hover放大1.02倍） |

## 3. 核心流程

用户打开页面 → 3D场景加载默认分子（CH4）→ 用户通过鼠标拖拽旋转视角、滚轮缩放 → 从左侧栏选择其他分子 → 原子和键平滑过渡动画 → 切换至键合模拟模式 → 点击第一个原子高亮 → 点击第二个原子创建/断裂键 → 右侧面板实时更新分子信息

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝黑 #0d1b2a
- **辅助高亮色**：冷桃色 #e0b1cb
- **数据信息色**：薄荷绿 #80cbc4
- **文字颜色**：白色 #e0e0e0
- **字体**：monospace风格
- **布局**：三栏布局（左140px + 中间全屏3D + 右280px）
- **原子颜色**：碳-深灰、氧-红色、氢-白色
- **键颜色**：默认浅灰#cccccc，悬停淡蓝#66b3ff
- **按钮**：渐变背景，轻微悬浮阴影，hover上移2px/0.2s，点击涟漪0.3s

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 左侧面板 | 渐变背景按钮，分子选择（H2O/CH4/CO2/NH3），键合模拟模式切换，hover上移动画 |
| 主页面 | 3D渲染区域 | 全屏深蓝黑背景，球棍分子模型，环境粒子，键悬停标签 |
| 主页面 | 右侧面板 | 分子式显示，原子/键数量统计，价态信息表格（交替背景，hover放大） |

### 4.3 响应式设计

- **≥1200px**：三栏布局（左140px + 中间3D + 右280px）
- **800-1200px**：隐藏左侧栏，分子选择改为顶部下拉菜单，保留右侧信息栏
- **<800px**：右侧信息栏折叠为可展开滑动面板

### 4.4 3D场景指导

- **环境**：深蓝黑空间，微弱环境粒子模拟溶剂
- **光照**：环境光+边缘高光，缩放时光照强度微调增强景深
- **相机**：透视相机，支持360度旋转（惯性0.5），滚轮缩放0.2-5倍
- **交互**：鼠标拖拽旋转，滚轮缩放，Raycaster检测原子/键悬停和点击
- **动画**：分子切换1.5s ease-in-out过渡，键形成0.5s延展动画，粒子特效
- **性能**：帧率≥40fps，粒子数≤300/帧
