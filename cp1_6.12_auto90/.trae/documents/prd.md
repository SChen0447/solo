## 1. 产品概述

虚拟涂鸦墙是一款基于地理位置和兴趣标签的城市探索应用，用户可以在城市地图的坐标点上留下带文字的彩色涂鸦，浏览者通过地图上的彩色气泡发现并互动。目标用户为城市探索者、旅行者和本地生活爱好者，产品价值在于以兴趣标签驱动的地理社交方式重新发现城市。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名密码登录 | 浏览地图涂鸦、发布涂鸦、按标签过滤 |

### 2.2 功能模块

1. **登录页**：暗色渐变背景、磨砂玻璃登录卡片
2. **地图主页**：Canvas城市道路网、涂鸦气泡标记、点击展开详情卡片
3. **右侧信息面板**：用户信息、标签过滤栏、关联涂鸦列表

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录页 | 登录卡片 | 磨砂玻璃质感卡片，用户名密码输入，渐变登录按钮 |
| 地图主页 | 道路网画布 | Canvas绘制模拟城市道路网（主干道4px/次干道2px白色半透明线） |
| 地图主页 | 涂鸦气泡 | 按标签颜色渲染气泡（电影金/美食翡翠/风景蓝），半径随浏览次数6-16px缩放 |
| 地图主页 | 详情卡片 | 点击气泡弹出毛玻璃卡片，显示文字内容+发布时间+关联涂鸦数量+展开按钮 |
| 信息面板 | 用户信息 | 圆形头像(60px)+紫色渐变边框+用户名 |
| 信息面板 | 标签过滤 | 5个胶囊标签按钮，选中时填充对应颜色+弹跳动画(scale 1→1.08→1) |
| 信息面板 | 关联涂鸦列表 | 水平卡片(80px高)，标签圆点+预览文字(前20字)+时间，悬停左移6px |

## 3. 核心流程

用户打开应用 → 登录页面输入用户名密码 → 登录成功进入地图主页 → 地图展示模拟城市道路网和涂鸦气泡 → 用户可拖拽/缩放地图浏览 → 点击气泡弹出详情卡片 → 右侧面板展示标签过滤和关联涂鸦 → 点击标签过滤不同类别涂鸦 → 点击关联涂鸦卡片定位到对应气泡

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝#0B1A2E → 紫黑#2A0A3D渐变（登录背景）、灰色#1F2937（地图背景）
- 强调色：紫色#8B5CF6、亮紫#A78BFA、粉色#EC4899
- 标签色：电影#F59E0B金色、美食#10B981翡翠、风景#3B82F6亮蓝
- 按钮风格：渐变圆角按钮，悬停上移+阴影加深
- 字体：思源黑体/系统无衬线字体，标题18px/正文14px/辅助12px
- 布局风格：全屏地图+右侧固定面板，卡片式信息展示

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录页 | 渐变背景 | 深蓝#0B1A2E到紫黑#2A0A3D线性渐变，全屏 |
| 登录页 | 登录卡片 | 宽380px，backdrop-filter:blur(12px)，边框#8B5CF6，圆角16px |
| 登录页 | 输入框 | 透明背景，聚焦时下边框+placeholder亮紫#A78BFA |
| 登录页 | 登录按钮 | 渐变#8B5CF6→#EC4899，悬停上移2px+阴影加深 |
| 地图主页 | 画布背景 | #1F2937深灰色 |
| 地图主页 | 道路线 | 白色半透明，主干道4px/次干道2px |
| 地图主页 | 涂鸦气泡 | 电影#F59E0B/美食#10B981/风景#3B82F6，半透明圆点，半径6-16px |
| 地图主页 | 详情卡片 | 宽260px，毛玻璃背景，0.3s ease-out展开，圆角12px |
| 信息面板 | 面板整体 | 宽280px，固定右侧置顶，背景#111827 |
| 信息面板 | 用户头像 | 60px圆形，2px紫色渐变边框 |
| 信息面板 | 标签按钮 | 胶囊形状，选中填充对应色+弹跳动画(1→1.08→1) |
| 信息面板 | 涂鸦卡片 | 高80px，背景#374151，圆角12px，悬停#4B5563+左移6px |

### 4.3 响应式设计

- 桌面端优先设计，地图画布占据屏幕左侧，信息面板固定右侧280px
- 地图画布通过 Canvas API 实现高性能拖拽/缩放，帧率≥55fps
- 涂鸦点批量加载(>100点)时首次渲染≤1.5秒

### 4.4 性能要求

- 地图拖拽和缩放帧率不低于55fps
- 超过100个涂鸦点时页面首次渲染时间不超过1.5秒
