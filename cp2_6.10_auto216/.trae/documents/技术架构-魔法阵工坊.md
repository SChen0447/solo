## 2. 技术描述

- **前端框架**：TypeScript + Vite@5
- **渲染引擎**：Canvas 2D API
- **工具库**：lodash、uuid
- **状态管理**：自研轻量状态模块（state.ts）
- **数据持久化**：浏览器 localStorage
- **动画系统**：requestAnimationFrame 帧循环 + 对象池粒子管理

## 5. 粒子系统设计

### 5.1 粒子对象池
- 预分配100个粒子对象，避免频繁GC
- 粒子复用通过 active 标志位控制
- 支持按特效类型分配不同粒子参数

### 5.2 粒子属性
| 字段 | 类型 | 说明 |
|------|------|------|
| active | boolean | 是否激活 |
| x, y | number | 当前位置 |
| vx, vy | number | 速度向量 |
| size | number | 粒子大小 |
| color | string | 粒子颜色 |
| alpha | number | 透明度 |
| life | number | 剩余生命周期 |
| maxLife | number | 最大生命周期 |

## 6. 性能优化策略

1. **对象池管理**：粒子对象预分配，复用而非销毁重建
2. **脏矩形渲染**：仅重绘变化区域（可选优化）
3. **帧率控制**：requestAnimationFrame 稳定60fps
4. **事件节流**：鼠标移动事件节流处理
5. **localStorage读写优化**：批量写入，避免频繁IO
