## 1. 产品概述

基于Web的3D蝴蝶翅膀鳞粉飘散与光影折射交互可视化应用，让用户在浏览器中以微距视角观察蝴蝶翅膀的鳞粉粒子在光线照射下产生的彩色折射与动态飘散效果。面向设计师、科学爱好者和教育领域用户，提供沉浸式的微观生物美学体验。

## 2. 核心功能

### 2.1 功能模块

1. **3D蝴蝶翅膀微距场景**：精细蝴蝶翅膀模型、半透明渐变材质、微观鳞片纹理、轨道控制交互
2. **鳞粉粒子系统**：1000个粒子持续发射、随机飘散旋转、视角依赖折射色彩、鼠标悬停交互
3. **动态光源与折射交互**：可拖拽方向光光源、光线方向影响翅膀折射色、光强影响粒子透明度
4. **视觉反馈与控制面板**：光照强度/粒子速度/旋转速度滑块、重置视角按钮、粒子暂停开关、FPS与粒子计数显示
5. **背景与性能管理**：渐变星空背景、闪烁星点、粒子自动回收机制、帧率监控

### 2.2 功能详情

| 模块名称 | 子功能 | 功能描述 |
|---------|-------|---------|
| 蝴蝶翅膀 | 几何建模 | 三个扁椭圆组合成一片翅膀，左右对称共两片 |
| 蝴蝶翅膀 | 结构色材质 | 根部深蓝渐变到外部紫色，透明度0.6-0.9，半透明效果 |
| 蝴蝶翅膀 | 鳞片纹理 | 叠加小三角形网格模拟鳞片排列，放大2倍可见细节 |
| 蝴蝶翅膀 | 轨道控制 | 鼠标拖拽旋转，阻尼缓动，0.5倍速，滚轮缩放 |
| 粒子系统 | 粒子发射 | 每片翅膀发射500粒子，共1000个，速率50粒/秒 |
| 粒子系统 | 粒子运动 | 半径0.02球体，随机飘散速度0.01-0.05，Y轴旋转0.01弧度/帧 |
| 粒子系统 | 折射色彩 | 视角与法线夹角从蓝→紫→橙渐变，onBeforeCompile自定义着色 |
| 粒子系统 | 悬停交互 | 鼠标悬停密集区时透明度0.3→0.8，加速飘散0.5秒 |
| 动态光源 | 可拖拽光源 | 小球体表示光源位置，限制在翅膀周围球面区域 |
| 动态光源 | 折射计算 | 入射角与法线点积计算色相偏移，蓝→红 |
| 动态光源 | 光强控制 | 光强0.3-1.0时粒子透明度线性0.2-0.8 |
| 控制面板 | 滑块控制 | 光照强度(0.3-1.0)、粒子速度(50-200/s)、旋转速度(0.1-1.0) |
| 控制面板 | 按钮控制 | 重置视角(1秒ease-in-out动画)、粒子暂停开关(0.2秒缩放动画) |
| 控制面板 | 状态显示 | FPS和粒子总数，每0.5秒刷新 |
| 背景环境 | 渐变星空 | 深蓝→暗紫径向渐变，80个闪烁白点(周期2秒正弦波) |
| 性能管理 | 粒子回收 | 超2000粒子时回收距离>5单位的粒子，0.5秒淡出 |
| 响应式设计 | 移动端适配 | <768px时控制面板折叠为浮动按钮，相机距离缩短20% |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝(#0a0a2e)、紫色(#6b21a8)、橙色(#f97316)渐变
- **辅色**：白色、青色点缀
- **控制面板**：毛玻璃效果，背景rgba(20,20,30,0.7)，模糊8px，圆角12px
- **滑块样式**：渐变轨道6px厚，白色圆形滑块20px直径带投影
- **字体**：现代无衬线字体，清晰可读
- **动效**：所有交互0.2-0.3秒ease平滑过渡

### 4.2 页面设计

| 区域 | UI元素 | 设计描述 |
|-----|--------|---------|
| 全屏3D画布 | WebGL Canvas | 无边框覆盖整个视口，渲染蝴蝶翅膀3D场景 |
| 右下角控制面板 | 半透明悬浮面板 | 深色毛玻璃，圆角12px，包含滑块、按钮、状态显示 |
| 控制面板-滑块区 | 三个滑块 | 渐变轨道，实时响应，标签+数值显示 |
| 控制面板-按钮区 | 两个按钮 | 重置视角按钮、暂停/继续开关，带动效 |
| 控制面板-状态区 | 状态文本 | FPS帧率、粒子总数，每0.5秒刷新 |
| 移动端 | 浮动按钮 | <768px时折叠为圆形按钮，点击展开垂直面板 |

### 4.3 响应式
- 桌面优先设计（1920x1080+）
- 断点768px：控制面板折叠为浮动按钮，相机距离缩短20%，面板垂直排列
- 触屏优化：支持触摸手势旋转和缩放

### 4.4 3D场景设计
- **环境**：深蓝→暗紫径向渐变星空背景，80个闪烁星点
- **光照**：单方向光（白色，强度1.0），可拖拽光源球体可视化
- **相机**：PerspectiveCamera，初始距离约5单位，近0.1远100
- **构图**：蝴蝶翅膀居中，微距视角，焦点在翅膀表面和鳞粉
- **交互**：轨道控制器（带阻尼）、光源拖拽、粒子悬停加速
- **后期**：半透明混合，无抗锯齿以保持性能
- **性能**：BufferGeometry/Points优化，目标30fps+，峰值2000粒子
