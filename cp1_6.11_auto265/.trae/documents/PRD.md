# 深海热液盲虾共生模拟 - 产品需求文档

## 1. 项目概述

### 1.1 产品定位
一款基于浏览器的深海生态模拟游戏，用户扮演深海生物学家，通过调节热液喷口环境参数，观察盲虾迁徙行为与共生菌群培养过程，探索极端环境下的生命共生机制。

### 1.2 目标用户
- 对深海生物学感兴趣的科普爱好者
- 学生和教育工作者
- 生态模拟游戏玩家

### 1.3 核心价值
- 沉浸式深海探索体验
- 可视化的生态共生原理展示
- 交互式的环境参数调节实验

## 2. 功能需求

### 2.1 深海环境模拟
- **背景渐变**：从深海永夜黑(#0a0f1c)到热液蓝(#1a3a5c)的全屏渐变背景
- **热液喷口**：中央矗立直径100px的SVG热液喷口，岩石纹理，顶端持续喷出烟雾粒子
- **烟雾粒子**：从中心白色(#ffffff)向外扩散为灰蓝(#5c8a8a)，透明度从0.8降至0
- **菌群斑块**：喷口周围随机分布15团直径30-50px的菌群斑块，颜色从深紫(#6a1b9a)到荧光绿(#00e676)渐变，边缘有微弱光晕
- **视角控制**：鼠标滚轮缩放、拖拽旋转视角

### 2.2 环境参数调节
- **温度滑块**：调节范围5°C - 95°C
- **pH值滑块**：调节范围2.0 - 9.0
- **硫化物浓度滑块**：调节范围0.1 - 5.0 mmol/L
- **实时反馈**：调节后烟雾颜色/数量、菌群生长速率/发光强度实时变化
  - 低温高硫 → 灰白色浓烟
  - 高温低硫 → 淡蓝色轻烟

### 2.3 盲虾行为模拟
- **初始数量**：20只盲虾随机游动
- **外观特征**：半透明弧形身体（长12px宽4px），颜色从淡粉(#f8bbd0)到乳白(#fff9c4)闪烁，头部两根触须，尾部生物发光点
- **趋温行为**：沿温度梯度自主游向最适区域（温度30°C-60°C、pH 4.0-6.0）
- **标记追踪**：点击任意盲虾标记，半透明淡黄色轨迹线实时记录路径

### 2.4 共生菌毯系统
- **菌毯形成**：盲虾停留在菌群斑块上超过3秒，开始刮取菌群形成体外共生菌毯
- **菌毯外观**：虾身体外围半透明放射状菌丝，颜色与所在菌群斑块一致，最大直径为虾身2倍
- **能力增强**：携带菌毯的盲虾游动速度提升20%，发光点亮度提升50%

### 2.5 菌群交换机制
- **触发条件**：两只带有不同颜色菌毯的盲虾相遇
- **交换动画**：两虾短暂纠缠，身体周围爆发彩色粒子圆环（直径60px，持续1.5秒）
- **颜色融合**：交换后双方菌毯颜色渐变融合，形成变种菌群

## 3. 界面设计

### 3.1 整体风格
- **色调**：深色调科幻深海风格
- **主背景**：深海永夜黑(#0a0f1c)到热液蓝(#1a3a5c)线性渐变
- **UI控件**：发光半透明设计，2px外发光描边(#00e5ff)，悬停缩放1.05倍加光晕

### 3.2 左侧控制面板
- **宽度**：200px
- **背景**：半透明黑色(rgba(0,0,0,0.7))
- **圆角**：12px
- **组件**：温度、pH、硫化物浓度三个发光滑块
- **滑块样式**：轨道颜色#37474f，发光圆形指针

### 3.3 右上角状态面板
- **内容**：盲虾数量、菌群覆盖率、共生配对次数
- **字体**：灰蓝色(#b0bec5)等宽字体
- **更新频率**：每帧更新

### 3.4 底部菌群图谱
- **展示形式**：环形图展示不同颜色菌群比例分布
- **动画效果**：每个扇形块带有微弱脉动动画

### 3.5 性能要求
- **帧率**：不低于30fps
- **响应时间**：点击标记盲虾响应不超过100ms

## 4. 技术要求

### 4.1 技术栈
- TypeScript
- React
- Vite 5
- Canvas 2D API
- Framer Motion
