## 1. 产品概述

个人音乐创作与混音工作站（Browser DAW）是一款基于浏览器的在线音乐制作工具，让用户无需安装专业软件即可通过虚拟音序器和采样器组合简单的旋律节奏。面向音乐爱好者、初学者和快速创意验证场景。

- 核心价值：零门槛音乐创作，即时听觉反馈，项目云端保存
- 目标用户：音乐爱好者、学生、内容创作者

## 2. 核心功能

### 2.1 功能模块

1. **主工作站页面**：音序器网格、混音面板、音色库、菜单栏、BPM控制
2. **音色选择系统**：8种预设音色卡片展示与分配
3. **音序编排系统**：16步x8轨网格音符编辑
4. **混音控制系统**：每轨音量、声像、静音、独奏
5. **项目管理系统**：保存/加载项目JSON数据
6. **播放与同步系统**：BPM调节、步进高亮、低延迟音频

### 2.2 功能详情

| 模块 | 功能点 | 描述 |
|------|--------|------|
| 音色库 | 8种预设音色 | 正弦波、锯齿波、方波、三角波、噪音、低音拨弦、踩镲、军鼓 |
| 音色库 | 音色卡片交互 | 悬停放大1.1倍+发光，选中边框变色，点击分配到当前音轨 |
| 音序器 | 16x8网格 | 每格32x32px，点击添加/移除音符，激活格半透明音色颜色 |
| 音序器 | 播放指示 | 黄色三角形当前步指示，每秒4步（默认BPM120） |
| 混音面板 | 音量控制 | 120px滑块，实时数值显示 |
| 混音面板 | 声像控制 | 圆形旋钮，拖动旋转角度控制左右声道 |
| 混音面板 | 静音/独奏 | 红色静音按钮、黄色独奏按钮 |
| 项目管理 | 保存项目 | JSON格式包含音色、网格、混音参数，绿色Toast提示 |
| 项目管理 | 加载项目 | 模态框列出项目名称（≤20字符）和创建时间 |
| 播放控制 | BPM调节 | 60-180范围，滑块+数字输入框同步 |
| 响应式 | 移动端适配 | <768px切换垂直堆叠布局 |

## 3. 核心流程

```mermaid
flowchart LR
    A["打开应用"] --> B["选择音轨"]
    B --> C["从音色库选择音色"]
    C --> D["在音序器网格点击添加音符"]
    D --> E["调节混音参数（音量/声像/静音/独奏）"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暗黑科技风，背景 #1A1A2E，面板 #2D2D3A，边框 #3A3A4A
- **强调色**：
  - 正弦波 #00E5FF（青）
  - 锯齿波 #FF6B6B（红）
  - 方波 #FFD93D（黄）
  - 三角波 #6BCB77（绿）
  - 噪音 #9D4EDD（紫）
  - 低音拨弦 #FF8C42（橙）
  - 踩镲 #E0AAFF（粉紫）
  - 军鼓 #06D6A0（薄荷绿）
- **功能色**：播放指示 #FFD700（金），静音 #FF1744，独奏 #FFD700
- **按钮样式**：圆角，悬停缩放/变色过渡 0.2s ease-out，按下 scale 0.95 0.1s
- **字体**：现代等宽字体 + 无衬线字体组合，营造专业DAW感
- **布局**：桌面端三栏（左280px混音 / 中音序器 / 右240px音色库），顶部40px菜单栏

### 4.2 页面设计概览

| 区域 | 模块 | UI元素 |
|------|------|--------|
| 顶部 | 菜单栏 | 保存/加载按钮，标题，40px高带底部阴影 |
| 左侧 | 混音面板 | 8轨垂直排列，每轨：音量滑块、声像旋钮、静音/独奏按钮 |
| 中央 | 音序器 | 16x8网格，步指示三角形，当前步高亮跳动 |
| 右侧 | 音色库 | 8张音色卡片（64x80px，圆角8px） |
| 右下 | BPM控制 | 200px滑块（刻度60/90/120/150/180）+ 数字输入 |

### 4.3 响应式
- 桌面优先（≥768px）：三栏水平布局
- 移动端（<768px）：垂直堆叠（上：菜单栏，中：音色库+音序器，下：混音面板+BPM控制）
- 触控优化：按钮最小44x44px触控区域

### 4.4 动效与微交互
- 音色卡片：hover scale(1.1) + box-shadow 发光，0.2s ease-out
- 所有可交互元素：active scale(0.95)，0.1s
- Toast通知：淡入淡出动画，3秒自动消失
- 音序器播放：当前步背景高亮跳动动画
- 滑块/旋钮：实时数值反馈
