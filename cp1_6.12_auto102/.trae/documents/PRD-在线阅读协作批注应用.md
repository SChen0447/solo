## 1. 产品概述

在线阅读协作批注应用，为多人线上读书会提供共同标注书籍段落、撰写批注、实时查看他人想法的协作工具。成员可在虚拟书页上点击段落添加批注，以气泡形式浮现在书页对应位置，支持点赞与评论互动。

- 核心价值：让阅读从单人行为转变为社交化、协作化的沉浸式体验
- 目标用户：读书会成员、学习小组、书友社群

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通成员 | 默认加入 | 浏览书籍、添加批注、点赞评论、切换书架视图 |

### 2.2 功能模块
1. **书架视图**：网格展示预设书籍封面，点击进入批注阅读
2. **阅读批注视图**：左右双页书本布局，左页显示文章段落，右页留白
3. **批注气泡**：圆形头像气泡浮现在书页上，显示批注预览，点击展开全文与评论
4. **批注输入**：选中段落后弹出毛玻璃输入框，提交后生成批注气泡
5. **点赞互动**：点赞按钮实时反馈点赞数

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 书架页 | 书籍网格 | 展示3本预设书籍封面卡片，支持hover上移动画 |
| 阅读页 | 双页书本 | 左右两页布局，左页文字，右页空白 |
| 阅读页 | 段落选择 | 点击文字行高亮选中，触发批注输入 |
| 阅读页 | 批注气泡 | 圆形头像气泡，悬停/点击展开详情 |
| 阅读页 | 批注输入框 | 毛玻璃效果，输入批注内容并发送 |
| 阅读页 | 评论区 | 气泡展开后显示评论列表与发表框 |
| 导航栏 | 顶部导航 | "书架"按钮切换视图，深灰背景 |

## 3. 核心流程

用户进入应用 → 默认显示书架视图 → 点击书籍封面进入阅读页 → 点击文字段落选中 → 弹出批注输入框 → 输入内容点击发送 → 批注气泡出现在对应位置 → 点击气泡展开查看全文和评论 → 可点赞或发表评论 → 点击"书架"返回书架视图

```mermaid

    A["进入应用"] --> B["书架视图"]
    B --> C["点击书籍封面"]
    C --> D["阅读批注视图"]

    E --> F["弹出批注输入框"]
    F --> G["输入并发送批注"]
    G --> H["生成批注气泡"]
    H --> I["点击气泡展开"]
    I --> J["查看全文/点赞/评论"]
    J --> K["点击书架返回"]
    K --> B
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖白背景 #F5F0E8，深灰导航 #37474F
- **强调色**：天蓝色 #4FC3F7（按钮、聚焦边框）
- **辅助色**：头像色板 #FF7043 / #66BB6A / #42A5F5 / #AB47BC / #FFA726
- **选中高亮**：淡黄色 #FFF9C4
- **按钮风格**：圆形发送按钮，柔和圆角 12px
- **字体**：系统无衬线字体
- **布局风格**：卡片式设计，CSS Grid + Flex 布局
- **动效风格**：subtle 阴影，hover 过渡 0.3s ease

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 书架页 | 导航栏 | 深灰背景 56px 高，水平居中，"书架"按钮 |
| 书架页 | 书籍卡片 | 280×380px，渐变色封面，box-shadow，hover 上移 4px |
| 阅读页 | 双页布局 | 左右两页模拟书本翻开效果 |
| 阅读页 | 文字段落 | 可点击行，选中时淡黄背景 |
| 阅读页 | 批注气泡 | 40px 圆形头像，显示3字预览，柔和阴影 |
| 阅读页 | 输入框 | 半透明毛玻璃，12px 圆角，边框 #B0BEC5 |
| 阅读页 | 发送按钮 | 圆形 #4FC3F7，点击反馈 |

### 4.3 响应式
- 桌面端优先设计
- 移动端 375px 宽度适配：
  - 批注气泡缩小为 30px 直径
  - 书架卡片变为纵向滚动
  - 双页布局调整为单页显示

### 4.4 性能要求
- 页面加载后 2 秒内显示书籍内容并完成初始批注渲染
- 批注渲染帧率不低于 30fps
- 批注数量超过 50 个时自动折叠最早批注以保持交互流畅
