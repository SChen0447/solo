## 1. 产品概述

基于时间胶囊概念的匿名情绪日记分享平台，让用户每天记录简短情绪日记（≤200字），系统自动分析情绪标签，匿名展示在动态时间线上。其他用户可以像捡漂流瓶一样浏览不同日期的情绪碎片，并留下温暖评论。
- 解决当代人情绪表达与匿名倾诉的需求，提供一个温暖、低压力的情绪记录空间
- 目标用户为需要情绪出口的年轻人群，产品价值在于情绪的匿名分享与共情连接

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名用户 | 自动分配匿名代号 | 浏览时间线、写日记、评论 |
| 已识别用户 | localStorage存储ID | 管理"我的胶囊"、编辑/删除自己的日记 |

### 2.2 功能模块

1. **时间线首页**：全景瀑布流视图，展示最近30天所有匿名日记卡片
2. **我的胶囊页**：垂直时间轴视图，展示当前用户的日记列表
3. **写日记浮层**：简短情绪输入，自动情绪分析
4. **日记详情弹窗**：完整内容展示、评论互动

## 3. 核心流程

**用户写日记流程**：用户点击写日记 → 输入情绪文字（≤200字）→ 系统自动分析情绪标签 → 匿名代号生成 → 日记保存至后端 → 时间线刷新展示新卡片

**用户浏览互动流程**：用户进入首页 → 浏览瀑布流卡片 → 点击卡片查看详情 → 点击"我想对ta说" → 输入评论 → 评论附加到卡片

**用户管理日记流程**：切换到"我的胶囊" → 查看时间轴 → 编辑或删除日记

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：柔和暖色系，背景采用昼夜渐变
- 情绪色带：开心#FFD700暖金、忧郁#6A5ACD薰衣草紫、暴躁#FF4500橙红、平静#87CEEB天蓝、焦虑#8B008B深紫
- 按钮风格：圆角16px，毛玻璃质感，300ms平滑过渡
- 字体：中文优先使用优雅衬线体，英文搭配柔和无衬线体
- 布局：卡片化设计，瀑布流+时间轴两种布局模式
- 匿名代号：随机"情绪形容词+动物名"组合（如"忧郁的考拉"）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 时间线首页 | 瀑布流区域 | 卡片宽180px，圆角12px，随机旋转-3°~3°，底部情绪色带6px高 |
| 时间线首页 | 背景 | 昼夜渐变：白天#E0F7FA→#B2EBF2，夜间#1A237E→#311B92 |
| 时间线首页 | 导航栏 | 顶部固定，毛玻璃背景，"时间线"和"我的胶囊"切换 |
| 日记详情弹窗 | 弹窗 | 半透明黑色模糊背景backdrop-filter:blur(8px)，spring动画开关 |
| 日记详情弹窗 | 卡片 | 居中显示，左上角匿名代号+时间，右下角"我想对ta说"按钮 |
| 日记详情弹窗 | 评论区 | 50字输入框，评论淡入动画 |
| 我的胶囊页 | 时间轴 | 垂直居中线，胶囊图标节点，左日期右卡片 |
| 我的胶囊页 | 删除对话框 | 红色"确认删除"+灰色"取消"，hover背景加深+0.5px下偏移 |

### 4.3 响应式设计

- 桌面端：瀑布流多列布局，卡片宽180px
- 移动端：单列竖向排列，卡片宽度100%，取消随机旋转
- 触屏优化：增大点击区域，滑动触底加载
