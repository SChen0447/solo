## 1. 产品概述
深海热泉生态系统演替与生物群落动态交互可视化——让用户扮演深海生态学家，通过调节热泉喷口参数（温度、硫化物浓度、喷发周期），实时观察从化学合成细菌爆发到完整深海热泉生态系统形成的演替过程。

## 2. 核心功能

### 2.1 功能模块
1. **全屏深海场景页**：3D热泉喷口+海底地形+2D生物群落叠加渲染
2. **参数控制面板**：温度/硫化物浓度/喷发周期三参数实时调节
3. **生态健康仪表盘**：环形进度条实时显示生态系统健康指数

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 深海场景页 | 3D热泉喷口 | Three.js实现喷口几何体(ConeGeometry)+烟柱粒子系统，粒子颜色白→灰蓝渐变，粒子大小3-6px |
| 深海场景页 | 海底地形 | Canvas 2D + Perlin噪声生成起伏丘陵纹理，深褐#3e2f28→墨绿#2b4a3b |
| 深海场景页 | 微生物席 | 半透明绿色斑点(5-15px随机散布)，透明度随温度30%→60% |
| 深海场景页 | 管虫群落 | Spline曲线生成细长弯曲管道(20-60px)，肉粉#ffaa77→深红#aa4422 |
| 深海场景页 | 贻贝床 | 密集深色椭圆点(8-12px, #1a3a5c)，间隔5-10px随机分布 |
| 深海场景页 | 盲虾 | 移动小点(3px, #ff8c00)，沿随机路径0.5-2px/帧移动 |
| 控制面板 | 温度滑块 | 范围50°C-400°C，刻度带渐变热力图色标(蓝-绿-黄-红) |
| 控制面板 | 硫化物浓度 | 滑块0-100%，环形进度条显示当前值 |
| 控制面板 | 喷发周期 | 下拉选择：持续/间歇/脉冲三种模式 |
| 仪表盘 | 健康指数 | 环形进度条，红(0%)→绿(100%)渐变，每2秒更新 |
| 交互反馈 | 参数过渡 | 参数调整后15秒内动画演替，framer-motion easeInOut缓动 |
| 交互反馈 | 悬停标签 | 鼠标悬停喷口显示温度标签(半透明黑底白字) |
| 交互反馈 | 群落过渡 | 粒子淡入淡出(不透明度0.2-1.0，持续2秒) |

## 3. 核心流程
用户进入全屏深海场景 → 右侧浮动控制面板调节参数 → 生态模拟逻辑计算群落密度/分布/健康指数 → 场景以15秒动画演替更新 → 仪表盘实时反映生态健康 → 循环调参观察

```mermaid
    "计算群落密度与分布" --> "Scene3D更新粒子/喷口"
    "计算群落密度与分布" --> "BiomeCanvas绘制生物群落"
    "计算群落密度与分布" --> "HUD更新健康指数"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深海永夜黑#0a0f1c → 热泉蓝#1a3a5c 渐变背景
- 辅助色：热泉亮白#ffffff、灰蓝#6a8aae、肉粉#ffaa77、深红#aa4422、橙黄#ff8c00
- 面板风格：半透明磨砂玻璃(backdrop-filter: blur(8px))，浮动右侧
- 字体：科学仪器风格，数据标签使用等宽字体
- 布局：全屏场景+右侧浮动控制面板+右下角仪表盘

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 深海场景页 | 全屏背景 | 深海黑→热泉蓝线性渐变，占满视口 |
| 深海场景页 | 控制面板 | 磨砂玻璃浮动面板，右侧定位，包含三个参数控件 |
| 深海场景页 | 仪表盘 | 右下角SVG环形进度条，红绿渐变 |
| 深海场景页 | 温度标签 | 鼠标跟随，半透明黑底白字 |

### 4.3 响应式设计
- 桌面优先设计，全屏沉浸式体验
- 控制面板固定右侧浮动，适配不同屏幕宽度

### 4.4 3D场景指导
- 环境：深海永夜氛围，仅热泉口提供微弱光照
- 灯光：热泉口点光源(暖白色)，微弱环境光(深蓝)
- 相机：正交俯视或略微倾斜透视，俯瞰热泉口及周围群落
- 构图：热泉喷口居中，生物群落环绕分布
- 交互：鼠标悬停喷口显示温度，参数调节实时更新
- 后处理：可选辉光效果增强热泉发光感
- 性能预算：粒子总数≤8000，保持60FPS
