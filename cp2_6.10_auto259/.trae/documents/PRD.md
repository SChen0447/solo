## 1. 产品概述

三轨音频波形可视化与音量混合调节应用，专为独立播客制作人设计，用于对齐、混合和可视化展示主持人、嘉宾和连线观众的三轨音频。

- 核心用途：播客后期制作中的音轨对齐、音量调节与实时波形可视化
- 目标用户：独立播客制作人、音频编辑人员

## 2. 核心功能

### 2.1 功能模块

1. **音频文件上传与播放控制**：三轨独立上传、统一播放控制、轨道静音
2. **音量混合与实时调节**：三轨独立音量滑块、波形透明度联动、实时音量柱状指示
3. **进度条定位与标记**：时间显示、点击跳转、刻度标记
4. **波形缩放与轨道间距**：水平缩放、垂直间距调节

### 2.2 功能详情

| 模块名称 | 子功能 | 功能描述 |
|---------|--------|----------|
| 音频上传 | 文件选择 | 三轨独立选择WAV/MP3文件 |
| | 缩略预览 | 上传后自动生成波形缩略图 |
| 播放控制 | 统一控制 | 播放/暂停/停止按钮，全局同步 |
| | 轨道静音 | 每轨独立静音，波形灰显 |
| 音量调节 | 垂直滑块 | 0-100范围，实时生效 |
| | 波形联动 | 音量高低对应波形透明度 |
| | 音量指示 | 左侧柱状能量条实时跳动 |
| 进度条 | 时间显示 | mm:ss格式的当前时间/总时长 |
| | 点击跳转 | 任意位置点击定位播放 |
| | 刻度标记 | 每隔5秒显示浅色刻度点 |
| 缩放调节 | 水平缩放 | 1x-10x波形缩放 |
| | 间距调节 | 轨道垂直间距紧凑/宽松 |

## 3. 核心流程

用户上传三轨音频 → 系统生成缩略波形 → 用户调节各轨音量 → 点击播放 → 实时绘制三色波形与音量柱 → 用户拖动进度条跳转/调节缩放 → 停止播放

```mermaid
flowchart LR
    A["上传三轨音频文件"] --> B["生成波形预览"]
    B --> C["调节音量/静音"]
    C --> D["点击播放"]
    D --> E["实时绘制波形与音量柱"]
    E --> F["进度条跳转/缩放调节"]
    F --> G["停止播放"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主背景色：深灰渐变 #1a1a2e → #16213e
- 波形背景：#0f3460
- 三轨道色：主持人 #ff6b6b、嘉宾 #51cf66、观众 #4dabf7
- 按钮样式：圆形大按钮，悬停放大1.1倍带光晕
- 滑块样式：垂直细条，弹性动画反馈
- 字体：现代无衬线字体，深色背景高对比度

### 4.2 页面布局

| 区域 | 位置 | 内容 |
|-----|------|------|
| 控制区 | 顶部 | 播放/暂停/停止按钮、文件上传入口 |
| 音量指示区 | 左侧 | 三轨垂直柱状音量条 |
| 波形展示区 | 中间（70%高度） | 三轨Canvas波形、轨道标签、音量滑块、进度条 |
| 调节区 | 底部 | 水平缩放滑块、轨道间距滑块 |

### 4.3 响应式设计

- 桌面端：标准上下结构布局
- 移动端（<768px）：波形区域高度缩小，底部调节区横向排列

### 4.4 交互动效

- 按钮：点击收缩弹起（framer-motion）
- 滑块：惯性滑动、弹性反馈
- 进度条：悬停变粗、时间气泡提示
- 整体：平滑过渡动画
