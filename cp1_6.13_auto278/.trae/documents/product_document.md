## 1. 产品概述

音纹·星语画布是一个协作式在线绘画社群平台，让用户通过浏览器在共享画布上绘制彩色光轨，系统根据画布上的颜色分布实时生成音乐旋律，创造集体视听共鸣体验。

- 解决问题：浏览器中缺乏一种通过协作绘画与实时音乐共鸣来创作集体视听作品的工具
- 目标用户：创意社区、艺术爱好者、音乐与视觉艺术交叉领域的创作者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 用户名/密码注册 | 登录后进入画布、绘制光轨、导出画布 |

### 2.2 功能模块

1. **登录/注册页**：用户身份验证，JWT鉴权
2. **画布主页**：协作绘画画布、颜色选择器、光轨绘制与粒子效果、音乐合成

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 登录/注册页 | 注册表单 | 用户名、密码输入，前端表单验证，调用注册API |
| 登录/注册页 | 登录表单 | 用户名、密码输入，前端验证，获取JWT令牌 |
| 画布主页 | 画布Canvas | 1200x900深紫到墨蓝径向渐变背景，星点闪烁，辐射网格，鼠标拖动绘制光轨 |
| 画布主页 | 光轨系统 | 12色调色板选取颜色，高斯模糊发光效果，2秒衰减动画，30个粒子跟随 |
| 画布主页 | 音乐合成 | Web Audio API，每5秒根据主色调RGB映射音符频率，琶音播放 |
| 画布主页 | 用户面板 | 右下角显示最近10位用户绘制记录，当前活跃人数，2秒轮询 |
| 画布主页 | 导出功能 | "导出画布"按钮，保存当前画布为PNG图片 |
| 导航栏 | 全局导航 | 毛玻璃风格，半透明背景，颜色跟随主色调变化，底部渐变边框 |

## 3. 核心流程

1. 用户注册/登录 → 获取JWT令牌 → 进入画布主页
2. 用户选择颜色 → 鼠标拖动绘制光轨 → 光轨带发光效果并缓慢衰减 → 粒子跟随飘散
3. 后端每5秒汇总活跃光轨颜色分布 → 计算主色调 → 返回前端
4. 前端根据主色调 → Web Audio API生成琶音旋律 → 每秒一个音符播放
5. 右下角面板实时显示活跃用户和绘制记录

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深紫(#1a0f2e)到墨蓝(#0d0b1a)径向渐变背景，12色光轨调色板
- 按钮风格：圆角毛玻璃按钮，半透明背景，发光边框
- 字体：标题使用 Orbitron（科技感），正文使用 Noto Sans SC（中文友好）
- 布局风格：全屏画布居中，左侧颜色选择器浮层，右下角信息面板，顶部毛玻璃导航栏
- 图标风格：lucide-react线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录/注册页 | 表单区域 | 毛玻璃卡片居中，深色背景，发光输入框边框 |
| 画布主页 | 画布区域 | 1200x900画布，径向渐变，辐射网格(rgba(255,255,255,0.05))，星点闪烁 |
| 画布主页 | 颜色选择器 | 12色圆形色块，悬停放大1.2倍+0.3px发光，浮动在画布左侧 |
| 画布主页 | 用户面板 | 右下角毛玻璃面板，显示最近10位用户+活跃人数 |
| 画布主页 | 导航栏 | 半透明(rgba(255,255,255,0.05))，backdrop-filter:blur(12px)，底部渐变边框3秒周期 |

### 4.3 响应式适配

- 桌面优先设计，画布默认1200x900
- 屏幕宽度<768px时，画布自动缩放至宽度100%，高度按比例缩小，最小高度600px
- 颜色选择器在小屏幕时折叠为底部横向色条

### 4.4 性能要求

- FPS稳定在55以上（使用requestAnimationFrame优化渲染）
- 后端API响应时间不超过50ms
- 粒子系统控制总数上限避免性能瓶颈
