## 1. 产品概述

「光诗·绘本走廊」是一款面向独立插画师的数字绘本展示与管理全栈Web应用，让插画师能够在浏览器中优雅地展示自己的漫画绘本作品。应用以纸质书般的翻页体验为核心特色，结合暖色调绘本风格的视觉设计，为读者提供沉浸式的阅读体验。

- 目标用户：独立插画师、绘本爱好者、漫画读者
- 核心价值：将数字绘本以接近纸质书的质感呈现，保留阅读的仪式感

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览故事列表、阅读绘本内容、调整阅读设置 |

### 2.2 功能模块
1. **故事列表页**：封面展示、故事信息、悬停交互、响应式布局
2. **绘本阅读页**：3D翻页动画、文本气泡渲染、阅读进度、工具栏操作

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 故事列表页 | 封面网格 | 3列响应式布局，展示封面小图(150x200px)、标题、作者 |
| 故事列表页 | 封面交互 | 毛玻璃蒙层效果，悬停放大(scale 1.05)，显示字数摘要 |
| 绘本阅读页 | 书本布局 | 480x680px居中，左右半页空白模拟书脊，中间1px虚线 |
| 绘本阅读页 | 3D翻页动画 | preserve-3d效果，Y轴旋转0.6秒，页面下角微翘，背面纸张纹理 |
| 绘本阅读页 | 文本气泡 | 圆角矩形渐变背景，弹入动画，位置可配置 |
| 绘本阅读页 | 阅读工具栏 | 翻页按钮、字体调节滑块(14-24px)、返回按钮 |
| 绘本阅读页 | 进度条 | 底部显示当前阅读进度 |
| 绘本阅读页 | 下一页预览 | 翻页中途显示低透明度的下一页缩略图和前5字预告 |

## 3. 核心流程

访客进入应用 → 浏览故事列表 → 点击感兴趣的故事封面 → 进入阅读页 → 通过翻页按钮阅读 → 调整字体大小 → 阅读完毕返回列表

```mermaid
    E --> F["调整阅读设置"]
    F --> G["返回列表"]

```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：#ff9a76（暖橙色）
- **辅助色**：#ffd3b6（浅桃色）
- **背景色**：#fdf5e6（米白色）
- **文字色**：#3e2723（深棕色）
- **按钮风格**：圆形翻页按钮(直径40px)，胶囊状返回按钮
- **字体**：serif, Georgia（系统圆体风格）
- **布局风格**：卡片式网格布局，居中书本阅读区
- **动效风格**：优雅的CSS 3D翻页、柔和的弹入动画、细腻的纸张纹理

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 故事列表页 | 封面网格 | 毛玻璃蒙层、悬停放大动画、渐变遮罩、字数摘要浮现 |
| 故事列表页 | 页面背景 | 米白背景、细腻纸张纹理(透明度0.05)、整体暖色调 |
| 绘本阅读页 | 书本容器 | 480x680px居中、米白背景、左右半页空白、书脊虚线 |
| 绘本阅读页 | 翻页动画 | 3D翻转(0.6s cubic-bezier)、下角微翘(rotateZ+translateY)、背面浅黄色纸张纹理 |
| 绘本阅读页 | 文本气泡 | 45度线性渐变(#ff9a76→#ffd3b6)、圆角20px、6px偏移阴影、弹入动画(scale 0.95→1.05→1) |
| 绘本阅读页 | 工具栏 | 毛玻璃背景(blur 10px, rgba(255,255,255,0.15))、圆形按钮组、字体滑块、胶囊返回按钮 |
| 绘本阅读页 | 进度条 | 底部线性进度条，与主色一致 |
| 绘本阅读页 | 下一页预览 | 低透明度缩略图、前5字预告、翻页中途短暂显示 |

### 4.3 响应式设计
- **桌面端**(≥768px)：故事列表3列，封面150x200px
- **平板端**(<768px)：故事列表2列，封面120x160px
- **移动端**(<480px)：故事列表1列，封面120x160px
- 阅读页书本容器自适应缩放，保持良好阅读体验

### 4.4 性能优化
- 翻页动画使用CSS 3D变换，启用GPU加速，保持60fps流畅
- 首屏内容(列表第一行)1.5秒内可见，采用懒加载策略
- 翻页过程中使用transform而非top/left避免布局抖动
