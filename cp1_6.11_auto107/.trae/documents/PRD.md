## 1. 产品概述

"命运之网"是一款在线虚拟占卜与命运编织应用，让用户化身神秘占卜师，通过塔罗牌、水晶球、星座罗盘等占卜工具生成个人命运图景，并将多名用户的命运丝线交织成一张动态社群命运网络，发现人与人之间隐藏的缘分联系。
- 目标用户：喜欢神秘学、占卜文化、社交互动的年轻群体
- 产品价值：将传统占卜体验数字化、游戏化，通过社群命运网络创造社交连接

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 占卜师 | 输入昵称和出生星座 | 进行占卜、生成命运星图、分享命运星图 |

### 2.2 功能模块
1. **主界面**：深紫色星空背景、闪烁星星粒子、浮动水晶球
2. **占卜选择**：塔罗牌、水晶占卜、星座罗盘三种方式
3. **塔罗牌占卜**：3x4牌阵、翻牌动画、22张大阿卡纳解读
4. **命运星图**：根据占卜结果和星座生成星图，5种排列模式
5. **命运预言**：诗意命运预言文字，含唯一哈希码

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主界面 | 星空背景 | 深紫色(#0d0b1a)背景，闪烁星星粒子(2-4秒随机周期) |
| 主界面 | 水晶球 | 浮动半透明水晶球(300px)，径向渐变玻璃质感，内部深蓝色雾气旋转(0.5度/秒) |
| 主界面 | 涟漪动画 | 点击水晶球后0.4秒扩散涟漪，弹出三种占卜方式选择卡片 |
| 占卜选择 | 选择卡片 | 塔罗牌(金色边框#d4af37)、水晶占卜(银框#c0c0c0)、星座罗盘(青铜框#cd7f32)，悬停放大1.1倍并柔光 |
| 塔罗牌占卜 | 牌阵 | 3x4牌阵，牌背星空图案，悬停倾斜15度并显示光晕 |
| 塔罗牌占卜 | 翻牌 | 0.6秒翻转动画(从左侧翻起，透视效果)，揭示22张大阿卡纳之一 |
| 塔罗牌占卜 | 牌面解读 | 矢量风格插画，50-100字中文寓意描述 |
| 塔罗牌占卜 | 命运编织 | 全部翻完后"编织命运"按钮，点击后牌面化作彩色光点飞向天空(1秒动画) |
| 身份注册 | 占卜师注册 | 输入昵称和出生星座(希腊符号表示) |
| 命运星图 | 星图生成 | 深蓝底色(#0a1128)，白色星点，5种排列模式(螺旋、波浪、放射、棋盘、随机) |
| 命运星图 | 预言展示 | 2段40-80字诗意命运预言，唯一哈希码用于分享 |

## 3. 核心流程

用户进入主界面 → 观看星空背景和浮动水晶球 → 点击水晶球触发涟漪动画 → 选择占卜方式(塔罗牌/水晶占卜/星座罗盘) → 进入对应占卜流程 → 完成占卜后注册占卜师身份 → 生成命运星图 → 获取命运预言和哈希码 → 分享/返回

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深紫色(#0d0b1a)为背景，深蓝(#0a1128)为星图底色
- 辅助色：金色(#d4af37)、银色(#c0c0c0)、青铜色(#cd7f32)
- 按钮风格：圆角半透明玻璃态按钮，悬停时发出柔光
- 字体：标题使用衬线字体(Cinzel)，正文使用无衬线字体
- 布局风格：全屏沉浸式，居中对称布局
- 图标/装饰：星座希腊符号、星星粒子、光点粒子

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主界面 | 星空背景 | 深紫色全屏背景，CSS动画闪烁星星(2-4秒随机周期)，粒子点缀 |
| 主界面 | 水晶球 | 300px直径，CSS径向渐变模拟玻璃质感，内部CSS旋转动画(0.5度/秒)，上下浮动动画 |
| 主界面 | 涟漪效果 | 点击后0.4秒扩散的环形涟漪CSS动画 |
| 占卜选择 | 选择卡片 | 三张半透明卡片，各色边框，悬停transform:scale(1.1)+box-shadow柔光 |
| 塔罗牌占卜 | 牌阵 | 3列4行网格，每张牌正面/背面，悬停perspective倾斜15度，光晕效果 |
| 塔罗牌占卜 | 翻牌动画 | CSS 3D transform翻转(0.6秒，从左翻起)，透视效果 |
| 塔罗牌占卜 | 编织命运 | 底部按钮，点击后牌面变为彩色光点(CSS动画1秒飞向天空) |
| 身份注册 | 注册表单 | 居中玻璃态表单，昵称输入框，星座选择器(希腊符号) |
| 命运星图 | 星图展示 | 深蓝底色Canvas/CSS，白色星点按5种模式排列，下方诗意预言文字，底部哈希码 |

### 4.3 响应式设计
- 桌面端优先设计，最小支持1024px宽度
- 水晶球和牌阵在大屏居中展示
- 移动端适配：牌阵缩小、卡片堆叠

### 4.4 动画性能要求
- 所有动画帧率保持在50fps以上
- 占卜结果生成延迟小于2秒
- 使用CSS动画和transform优先，减少JavaScript动画计算
