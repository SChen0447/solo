## 1. 产品概述

珊瑚礁健康模拟器是一款面向珊瑚礁生态学家和海洋保护区管理者的3D交互式浏览器应用。它将抽象的水温数据与珊瑚白化现象转化为可旋转、可缩放、可点击的沉浸式3D水下体验，帮助用户直观理解珊瑚礁生态在不同水温条件下的变化过程。

- 核心目标：将传统数据图表转化为空间化、可视化的3D交互体验，辅助珊瑚礁健康评估与生态保护决策
- 目标用户：珊瑚礁生态学家、海洋保护区管理者、海洋环境教育工作者

## 2. 核心功能

### 2.1 功能模块

1. **3D海底场景与珊瑚群落**：程序化生成海底地形，分布至少三种形态的珊瑚（分枝状、块状、叶片状），支持水温驱动的颜色渐变与白化指数计算
2. **浮动控制面板**：水温调节滑块、时间模拟播放按钮、珊瑚信息展示面板，支持毛玻璃效果与滑入动画
3. **统计折线图**：实时绘制全场景平均白化指数随时间变化曲线，支持CSV基线数据导入与虚线对比显示
4. **环境与生机指标**：顶/底双色环境光、动态鱼群游动，鱼群数量随白化指数动态调整
5. **全局快捷键**：H键切换健康/白化视图、R键重置、S键截图、空格键暂停/恢复

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深海蓝紫渐变（背景#0a1a2a），营造沉浸式深海氛围
- **点缀色**：科技荧光蓝#00e5ff，用于UI控件边框、发光效果、焦点指示
- **按钮风格**：圆角矩形，悬停时scale(1.05) + 发光阴影（box-shadow: 0 0 12px #00e5ff80）
- **字体**：白色#ffffff，细体(light weight)，主标题28px，正文14px
- **布局**：3D全屏场景 + 左侧浮动控制面板 + 右下角折线图 + 顶部居中标题
- **动画风格**：ease-out缓动曲线，控制面板滑入0.3秒，颜色过渡0.8秒

### 4.2 页面设计概览

| 区域 | 模块 | UI元素 |
|------|------|--------|
| 全屏背景 | 3D场景 | Three.js Canvas，海底地形、珊瑚群落、鱼群、环境光 |
| 顶部中央 | 页面标题 | "珊瑚礁健康模拟器" 28px白色细体，带渐变下划线 |
| 左侧 | 浮动控制面板 | 半透明毛玻璃rgba(20,40,60,0.85)，圆角16px，宽度260px，滑入动画 |
| | 水温滑块 | 渐变刻度条，范围22-36°C，当前值显示 |
| | 播放按钮 | 三角播放/双竖线暂停图标，荧光蓝描边 |
| | 信息面板 | 三行数据：物种名称、白化指数%、共生藻密度 |
| 右下角 | 折线图窗口 | 深蓝透明背景，圆角8px，240×120px，Chart.js绘制 |
| 点击反馈 | 波纹动画 | 点击处圆形扩散，半径0→40px，透明度0.6→0 |

### 4.3 响应式适配
- **PC宽屏（>1200px）**：三排珊瑚布局，控制面板260px完全展开
- **平板横屏（768-1200px）**：两排珊瑚，控制面板宽度不变
- **手机竖屏（<768px）**：单排珊瑚，控制面板折叠为图标按钮，点击展开
- 所有触控区域最小44×44px，支持双指缩放旋转

### 4.4 3D场景指导
- **环境与氛围**：深海散射光，顶/底半球光模拟海水深度光照衰减，caustics水面焦散纹理平铺
- **光照设置**：AmbientLight基础环境光 + HemisphereLight(#87CEEB / #0a2a4a)
- **相机动画**：PerspectiveCamera初始位置(0, 15, 25)，OrbitControls支持自由旋转缩放，阻尼开启
- **构图与焦点**：珊瑚群落在中心区域(±15, ±10)散布，相机俯视45°角
- **交互与动画**：珊瑚颜色0.8秒平滑插值，鱼群帧动画游动，波纹CSS动画
- **性能目标**：50珊瑚+80鱼稳定>45fps，珊瑚使用BufferGeometry合并渲染

### 4.5 性能指标
- 帧率：≥45fps（50珊瑚+80鱼场景）
- UI响应：菜单切换≤100ms
- 颜色过渡：0.8秒平滑插值（CSS transition / Three.js Color.lerp）
