## 1. 产品概述

音频段落提取器是一款面向独立音乐人和播客制作者的浏览器端工具，用于快速从长录音中自动识别人声片段、标记段落并导出独立音频文件。它解决了传统音频编辑软件（Audacity、Logic Pro）手动切分耗时费力的痛点，将30分钟的重复劳动压缩到几分钟。

- 目标用户：独立音乐人、播客主播、有声书创作者
- 核心价值：一键静音检测、可视化分段标记、批量导出音频片段

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 创作者 | 无需注册，直接使用 | 上传音频、调整阈值、编辑片段、导出文件 |

### 2.2 功能模块

1. **波形显示区**：实时波形渲染、阈值线、静音遮罩、片段色块标记
2. **阈值控制面板**：灵敏度滑块（-40dB 到 0dB）、实时静音检测
3. **片段列表面板**：片段编号、标签、时长、删除操作、跳转定位
4. **导出工具栏**：批量导出配置、格式选择（WAV/MP3）、ZIP打包下载

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 文件上传区 | 拖拽或点击上传 .wav/.mp3 文件（最大20MB） |
| 主页面 | 波形显示区 | Canvas 渲染波形、阈值线、静音遮罩、片段色块 |
| 主页面 | 阈值控制 | 左侧滑块调节灵敏度，实时更新静音检测结果 |
| 主页面 | 片段色块栏 | 彩色色块标记有声片段，可拖拽调整边界、点击输入标签 |
| 主页面 | 片段列表面板 | 垂直列表展示所有片段，点击跳转并高亮 |
| 主页面 | 导出工具栏 | 顶部工具栏，"全部导出"按钮触发配置对话框 |
| 主页面 | 导出对话框 | 选择导出格式、是否合并标签到文件名、确认下载 |

## 3. 核心流程

用户上传音频文件 → 系统解码并渲染波形 → 自动按阈值检测静音/有声片段 → 用户拖拽调整片段边界或输入标签 → 点击导出按钮 → 选择格式和命名规则 → 浏览器自动下载 ZIP 压缩包。

```mermaid

    A["上传音频文件"] --> B["解码获取波形数据"]
    B --> C["Canvas 渲染波形"]
    C --> D["阈值检测静音片段"]
    D --> E["生成彩色片段色块"]
    E --> F["用户拖拽调整/输入标签"]
    F --> G["点击全部导出"]
    G --> H["配置导出选项"]
    H --> I["打包 ZIP 下载"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题，主背景 #1a1b2f，次级背景 #2d2d3d
- **强调色**：波形曲线 #4fc3f7，主按钮 #6c5ce7，删除操作 #e74c3c
- **片段色板**：按顺序循环 #ff6b6b, #4ecdc4, #ffe66d, #a29bfe, #fd79a8
- **圆角规范**：控件 8px，对话框 12px，色块 6px
- **阴影规范**：box-shadow: 0 2px 8px rgba(0,0,0,0.3)
- **过渡动画**：0.3s ease-in-out，按钮点击 transform: scale(0.95)
- **字体规范**：系统默认无衬线字体，标签文字 13px

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主页面 | 波形显示区 | 100%宽，320px高，#1a1b2f 背景，淡蓝色波形曲线 |
| 主页面 | 阈值线 | #888 灰色水平线，左侧红色滑块（#ff6b6b）调节 |
| 主页面 | 静音遮罩 | 半透明红色 rgba(255,0,0,0.15) 覆盖静音区域 |
| 主页面 | 片段色块栏 | 高度24px，彩色圆角色块，可拖拽边界带垂直虚线辅助线 |
| 主页面 | 片段列表面板 | 右侧280px宽，#2d2d3d 背景，可滚动列表 |
| 主页面 | 导出工具栏 | 顶部56px高，#222233 背景，紫色导出按钮 |
| 主页面 | 导出对话框 | #1e1e2e 背景，12px圆角，半透明遮罩覆盖 |

### 4.3 响应式

- Desktop-first 设计
- 宽度低于 768px 时：右侧片段面板折叠为底部可展开浮动工具栏，波形高度自适应为 200px
- 触摸设备优化：拖拽区域增大，按钮最小触控尺寸 44px

### 4.4 性能指标

- 20MB 以内音频波形渲染 ≤ 1 秒
- 阈值调整后静音检测更新 ≤ 200 毫秒
- 所有动画保持 60fps
