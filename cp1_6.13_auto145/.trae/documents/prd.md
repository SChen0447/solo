## 1. 产品概述

谜境·词踪是一款中国水墨画风格的文字解谜Web应用，用户在虚拟迷雾森林中探索，解开历史典故、成语和冷知识谜题，解锁水墨配图与短文介绍，所有记录汇聚于个人图鉴。

- 目标用户：喜爱中国传统文化、文字游戏和探索类体验的用户
- 核心价值：以游戏化方式学习历史典故，在沉浸式水墨美学中获得知识与成就感

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 浏览地图、解谜、查看图鉴 |

### 2.2 功能模块

1. **迷雾森林地图页**：全屏地图，6-8块迷雾区域，水墨松树背景，悬停交互，谜题弹窗入口
2. **个人图鉴面板**：右侧滑入面板，已解谜条目卡片展示，支持删除和详情查看

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 迷雾森林地图 | 地图区域 | 6-8块不规则多边形迷雾区域，悬停变透明，点击弹出谜题 |
| 迷雾森林地图 | 松树装饰 | 7-9棵水墨松树剪影，随机位置和大小 |
| 迷雾森林地图 | 图鉴入口按钮 | 右下角固定按钮，显示收集进度 |
| 谜题弹窗 | 题目展示 | 毛笔字体编号、文字谜题、卷轴输入框、朱砂印章提交按钮 |
| 谜题弹窗 | 解谜反馈 | 正确后迷雾消散动画、水墨配图浮现、短文介绍淡入 |
| 个人图鉴 | 收藏列表 | 卡片展示配图缩略图、题目摘要、解锁日期，支持删除 |
| 个人图鉴 | 详情查看 | 点击卡片查看完整配图和介绍 |

## 3. 核心流程

用户打开页面 → 看到迷雾森林地图 → 悬停迷雾区域查看提示 → 点击迷雾区域弹出谜题 → 输入答案并提交 → 后端验证答案 → 正确：迷雾消散动画 + 水墨配图浮现 + 短文介绍 → 条目添加到图鉴 → 可随时查看图鉴回顾

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：墨黑（#1a1a1a）、宣纸米黄（#f5e6d0）、朱砂红（#c23b22）、青绿（#2e8b57）
- 按钮风格：朱砂印章圆形按钮（直径40px），复古金色圆形图鉴按钮（直径50px）
- 字体：毛笔风格标题字体，正文使用衬线中文字体
- 布局风格：全屏自适应地图布局，最小宽度1024px，弹窗居中覆盖
- 图标/装饰：水墨松树剪影、金色虚线边框、卷轴输入框

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 迷雾森林地图 | 地图背景 | 墨绿色渐变背景，半透明灰色迷雾(0.6)，水墨松树剪影(80-120px) |
| 迷雾森林地图 | 迷雾区域 | 不规则多边形，悬停透明度0.2+金色虚线边框，点击弹出弹窗 |
| 迷雾森林地图 | 图鉴按钮 | 右下角固定，直径50px圆形，复古金色纹理，显示收集数"3/8" |
| 谜题弹窗 | 整体 | 350x450px，居中，宣纸纹理背景(浅米色噪点) |
| 谜题弹窗 | 标题区 | 毛笔字体编号"谜·壹" |
| 谜题弹窗 | 题目区 | 40-80字中文诗句/成语描述 |
| 谜题弹窗 | 输入框 | 淡黄色底色+深棕色边框，古风卷轴样式 |
| 谜题弹窗 | 提交按钮 | 圆形朱砂印章，直径40px，红色#c23b22，白色"答"字 |
| 解谜成功 | 迷雾消散 | 透明度0.6→0，持续1.5秒，CSS缓动 |
| 解谜成功 | 配图浮现 | 200x200px水墨配图，边缘墨水晕染，GSAP fromTo淡入2秒 |
| 解谜成功 | 短文介绍 | 140-200字，淡入动画2秒 |
| 个人图鉴 | 面板 | 右侧滑入，宽350px，高90%视口，深蓝色渐变背景 |
| 个人图鉴 | 卡片项 | 配图缩略图80x80px，题目摘要(20字截断)，解锁日期 |

### 4.3 响应式适配

- 桌面优先设计，最小宽度1024px
- 1400x900和1920x1080分辨率下地图区域等比例缩放
- 迷雾区域使用百分比定位，适配不同屏幕尺寸

### 4.4 动画规范

- 迷雾消散：CSS关键帧，透明度0.6→0，1.5秒，ease-out缓动
- 配图浮现：GSAP fromTo，opacity 0→1 + scale 0.8→1，2秒
- 短文淡入：CSS transition，opacity 0→1，2秒
- 图鉴面板：CSS transform，translateX(100%)→translateX(0)，0.3秒
- 性能目标：地图初始加载≤2秒，API响应≤500ms，动画帧率≥55fps
