## 1. 产品概述
数字植物园珍稀植物3D生长模拟应用，游客可沉浸式观察不同气候带下珍稀植物的生长过程，通过调节光照和水分参数，实时查看植物形态与颜色的动态变化。

- 主要用途：教育展示、科普互动、数字植物园体验
- 目标用户：植物园游客、自然爱好者、学生群体
- 产品价值：将抽象的植物生长过程可视化、可交互化，提升科普教育的趣味性和沉浸感

## 2. 核心功能

### 2.1 功能模块
1. **3D植物展示模块**：三种珍稀植物（仙人掌、兰花、蕨类）的3D建模与渲染，支持平滑切换动画
2. **生长阶段模拟模块**：植物从幼苗到成熟的全阶段形态变化，包含高度、粗细、叶片数量、开花程度等参数
3. **环境参数交互模块**：光照强度、水分湿度的实时调节，影响植物形态与颜色
4. **粒子特效模块**：开花时的彩色粒子飘散效果
5. **场景氛围模块**：渐变天空盒、草地地面、石头装饰、环绕光晕点
6. **控制面板模块**：半透明毛玻璃风格UI，响应式适配

### 2.2 功能详情
| 模块名称 | 功能点 | 功能描述 |
|---------|--------|----------|
| 植物展示 | 植物切换 | 下拉框选择植物，旧植物0.5秒渐隐，新植物0.8秒渐显 |
| 植物展示 | 默认展示 | 初始显示仙人掌，生长阶段20% |
| 生长模拟 | 进度调节 | 0-100%进度条控制，GSAP 0.3秒平滑过渡 |
| 生长模拟 | 形态变化 | 仙人掌高度1→4单位，直径0.6→1.2单位，颜色浅绿→深绿 |
| 环境交互 | 光照影响 | 植物朝向光源偏转±30°，受光面提亮，背光面变暗，光照不足叶子下垂 |
| 环境交互 | 水分影响 | 水分不足叶片卷曲缩小20% |
| 开花特效 | 条件触发 | 生长>60%，光照>50%，水分>50%时触发 |
| 开花特效 | 粒子系统 | 每2秒生成20个彩色粒子，1.5秒向上飘散渐隐 |
| 场景氛围 | 天空背景 | 淡蓝到淡橙渐变，切换植物时2秒渐变到对应气候带色 |
| 场景氛围 | 地面装饰 | 草地平面+10颗随机小石头+8个脉动光晕点 |
| UI面板 | 控制面板 | 半透明毛玻璃风格，左侧240px，小屏折叠为浮动按钮 |
| UI面板 | 信息显示 | 右下角显示植物名称和生长百分比 |

## 3. 核心流程
用户进入应用 → 默认展示仙人掌（生长20%）→ 用户可调节光照/水分/生长进度 → 观察植物实时变化 → 可切换至兰花/蕨类 → 条件满足时触发开花粒子特效

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：根据植物气候带动态变化（仙人掌暖黄#f4d03f、兰花淡紫#af7ac5、蕨类翠绿#27ae60）
- **背景色**：淡蓝到淡橙渐变天空盒
- **控制面板**：毛玻璃效果 `rgba(255,255,255,0.15)`，模糊10px，圆角8px
- **字体**：现代无衬线字体，白色带文字阴影
- **控件风格**：滑块渐变填充轨道，下拉框平滑展开，所有交互ease-in-out 0.2秒缓动

### 4.2 页面布局
| 区域 | 元素 | 样式说明 |
|------|------|----------|
| 全屏背景 | 3D场景画布 | z-index: 0 |
| 左侧 | 控制面板 | 240px宽，毛玻璃，z-index: 10 |
| 右下角 | 植物信息 | 14px白色字，带阴影，z-index: 10 |
| 小屏 | 浮动按钮 | 可拖动，点击展开面板 |

### 4.3 响应式设计
- 桌面端：控制面板左侧固定展开
- 平板/移动端：面板折叠为浮动按钮，点击展开，支持拖动
- 所有控件支持触摸操作

### 4.4 3D场景设计
- **环境**：渐变天空盒（淡蓝→淡橙），草地地面（#4a7c3f，半径10单位）
- **灯光**：主方向光+环境光+8个植物周围半透明脉动点光源（强度0.2-0.5）
- **相机**：透视相机，轨道控制器，支持旋转缩放
- **装饰**：10颗随机分布小石头（IcosahedronGeometry，0.1-0.3单位）
- **性能预算**：单株植物<800三角形，粒子数<50，FPS>30
