## 1. 产品概述
对话卡片是一款将日常文字对话转化为限量数字艺术卡片的全栈Web应用，解决用户无法将文字中的情感和关键词视觉化、收藏化和社交化的问题。
- 核心价值：将普通文字转化为具有艺术价值和收藏属性的数字卡片，通过情感分析和关键词提取赋予文字独特的视觉表达
- 目标用户：喜欢记录生活、追求个性化表达、热衷于数字收藏和社交分享的年轻用户群体

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（演示模式） | 创建卡片、收藏、点赞、评论、添加标签、浏览画廊 |

### 2.2 功能模块
1. **卡片生成页**：文字输入、关键词提取展示、情感分析配色、Canvas实时渲染预览、卡片保存
2. **个人画廊页**：瀑布流卡片展示、卡片悬浮动效、标签筛选、卡片详情查看
3. **探索页**：所有用户卡片展示、按共鸣标签筛选
4. **卡片详情**：点赞（红心动画）、评论系统、关注用户、共鸣标签展示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 卡片生成页 | 文字输入区 | 最多200字文本框，实时字数统计，生成按钮 |
| 卡片生成页 | 关键词展示区 | 自动提取3-5个关键词，以标签形式展示 |
| 卡片生成页 | Canvas预览区 | 实时渲染卡片，包含标题、正文、时间戳、编号、二维码、几何背景 |
| 个人画廊页 | 瀑布流布局 | 三列/两列/单列响应式布局，卡片悬浮放大1.05倍+阴影 |
| 个人画廊页 | 标签筛选 | 点击共鸣标签筛选同标签卡片 |
| 卡片详情 | 点赞功能 | 红心从中心向外扩散再收缩动画，持续0.5s |
| 卡片详情 | 评论功能 | 圆角8px评论框，背景#F5F5F5，发布按钮主题色 |
| 卡片详情 | 共鸣标签 | 最多3个标签，圆角矩形，背景#7C4DFF，白色文字12px |

## 3. 核心流程
用户输入一段文字 → 系统提取关键词并分析情感 → 根据情感匹配配色方案和图形模板 → Canvas渲染生成对话卡片 → 用户保存卡片到画廊 → 用户可添加共鸣标签 → 其他用户可点赞、评论、关注

```mermaid

    A["用户输入文字"] --> B["关键词提取"]
    B --> C["情感分析"]
    C --> D["匹配配色与模板"]
    D --> E["Canvas渲染卡片"]
    E --> F["保存到画廊"]
    F --> G["添加共鸣标签"]
    G --> H["社交互动(点赞/评论)"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，主背景#121212，卡片底色#1E1E1E到#2D2D2D渐变
- **主题色**：紫色系#7C4DFF（主色）、#B388FF（强调色）、#03DAC5（成功色）
- **按钮样式**：圆角设计，hover时变亮10%，点击有0.2s微交互
- **字体**：标题衬线体24px，正文无衬线体16px，标签12px
- **布局风格**：左侧竖排tabs（宽120px，选中态左侧3px粗线#7C4DFF），中间内容区最大1200px居中
- **动效**：卡片进入视口从下向上滑动渐入（0.6s cubic-bezier(0.22, 1, 0.36, 1)），点赞红心扩散收缩（0.5s）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 全局 | 顶部导航栏 | 高64px，背景#1E1E1E，左侧Logo 22px粗体，右侧圆形头像48px |
| 全局 | 左侧Tabs | 竖排，宽120px，选中态左侧3px粗线#7C4DFF |
| 卡片生成 | Canvas卡片 | 几何抽象背景（圆形阵列/三角形镶嵌），标题24px衬线体，正文16px无衬线体，底部时间戳+D-XXXXX编号，左下角二维码 |
| 画廊 | 瀑布流卡片 | 悬浮放大1.05倍+阴影过渡0.3s ease，进入视口滑入动画 |
| 卡片详情 | 共鸣标签 | 圆角矩形，背景#7C4DFF，白色文字12px，最多3个 |
| 卡片详情 | 评论框 | 圆角8px，背景#F5F5F5，发布按钮#7C4DFF hover变亮10% |

### 4.3 响应式设计
- **桌面端（1200px以上）**：三列瀑布流，左侧竖排tabs正常显示
- **平板端（768-1199px）**：两列瀑布流，左侧竖排tabs正常显示
- **移动端（小于768px）**：单列瀑布流，隐藏左侧tabs，改为顶部横向滚动tab条

### 4.4 性能要求
- Chrome浏览器瀑布流渲染保持45FPS以上
- 首次加载20张样例卡片时间小于1.5秒
