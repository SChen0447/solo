## 1. 产品概述

「光痕·符文回响」是一款基于浏览器的回合制策略游戏，玩家在由发光符文构成的8x8动态棋盘上，通过点击翻转符文激活元素之力（火、水、风、土），触发涟漪特效与音效，当三个及以上同色符文成线时爆裂消失并产生元素共鸣波伤害对手，以消灭对手全部血量碎片为目标。

- 目标用户：休闲策略游戏玩家、视觉特效爱好者
- 核心价值：将策略连线玩法与炫酷元素特效深度融合，提供沉浸式视觉体验

## 2. 核心功能

### 2.2 功能模块

1. **游戏主界面**：8x8符文棋盘、双方血量碎片条、回合数与得分显示、元素裂变特效

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 游戏主界面 | 棋盘区域 | 8x8方形网格，每格60px，间隔4px，整体居中微旋转2度，格子内发光符文缓慢呼吸 |
| 游戏主界面 | 符文系统 | 四种元素符文（火-三角形/红#ff6b6b、水-波浪线/蓝#48dbfb、风-螺旋/绿#00b894、土-菱形/褐#a29bfe），点击翻转180度换色 |
| 游戏主界面 | 连线检测 | 翻转后自动检测水平、垂直、对角线同色连线（≥3个），触发爆裂粒子特效与波纹扩散 |
| 游戏主界面 | 血量碎片条 | 10块发光碎片（每块10%），受击飞散再聚合，完全消失则败北 |
| 游戏主界面 | 顶部信息栏 | 回合数（发光脉冲数字）、双方得分（圆形图标+数字） |
| 游戏主界面 | 元素裂变 | 每10回合全屏闪白，未连线符文随机洗牌换色 |
| 游戏主界面 | 胜利特效 | 金黄光柱从棋盘中心升起，粒子上升持续2秒 |

## 3. 核心流程

玩家点击格子翻转符文 → 翻转动效（0.4秒绕X轴旋转180度换色）→ 连线检测（水平/垂直/对角线≥3同色）→ 连线符文依次爆裂（粒子飞散+波纹扩散）→ 对手血量碎片飞散→ AI回合（延迟0.8-1.2秒决策）→ AI翻转符文 → 同样检测连线 → 循环至碎片条清零 → 胜利光柱

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝紫#0b0e27至墨黑#0a0a0a径向渐变背景
- 强调色：青蓝#48dbfb（边框/回合数）、各元素本色
- 符文风格：发光、呼吸动画、元素形态差异化
- 布局：全屏Canvas，棋盘居中，信息栏右上角固定
- 动效：翻转、爆裂、波纹、碎片飞散聚合、元素裂变、胜利光柱

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 游戏主界面 | 背景 | 径向渐变#0b0e27→#0a0a0a，棋盘外圈发光边框#48dbfb透明度0.3脉动2秒 |
| 游戏主界面 | 棋盘 | 8x8网格，格子60px间隔4px，整体旋转2度，符文呼吸缩放0.9-1.05周期3-5秒 |
| 游戏主界面 | 符文 | 火三角形红#ff6b6b、水波浪线蓝#48dbfb、风螺旋绿#00b894、土菱形褐#a29bfe |
| 游戏主界面 | 碎片条 | 10块发光碎片#ff6b6b，飞散半径30px旋转360度0.5秒，聚合0.8秒 |
| 游戏主界面 | 信息栏 | 回合数发光脉冲1.5秒20px#48dbfb，得分圆形图标+数字 |
| 游戏主界面 | 特效 | 爆裂粒子20个60px半径0.6秒，波纹200px/秒1秒衰减，胜利光柱金黄500px2秒 |

### 4.3 响应式

- 桌面优先：格子60px
- 宽度<768px：格子缩小至40px，符文字号相应缩小
- 全屏Canvas自适应视口，无滚动条

### 4.4 性能要求

- 保持稳定60FPS
- 粒子系统最大同时活跃粒子≤200
- AI决策时间≤1.5秒
