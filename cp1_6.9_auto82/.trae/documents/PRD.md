## 1. 产品概述

动态光合作用生态系统模拟器 - 用户扮演植物学家，在 3D 空间中放置虚拟植物，通过调节光照、CO2、水分等环境参数，观察植物生长状态变化和氧气释放效果，形成动态微型生态可视化画面。

- 核心价值：教育性可视化工具，直观展示光合作用与环境因素的关系
- 目标用户：学生、教育工作者、生物爱好者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 访客用户 | 无需注册 | 完整使用所有模拟功能 |

### 2.2 功能模块
1. **3D 场景模块**：星空背景悬浮草地、轨道相机控制、植物渲染
2. **植物管理模块**：三种植物模型、动态添加/移除、生长状态管理
3. **环境控制模块**：光照强度、CO2 浓度、水分参数滑块调节
4. **气泡粒子模块**：最佳环境下氧气气泡生成与飘移动画
5. **UI 面板模块**：参数控制面板、帧率显示、响应式布局

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 3D 场景 | 圆形悬浮草地、星空背景、预设三株植物、鼠标拖拽旋转/滚轮缩放 |
| 主页面 | 参数面板 | 右侧垂直滑块：光照(0-100lux)、CO2(200-2000ppm)、水分(0-100%) |
| 主页面 | 状态显示 | 右上角实时帧率、参数面板提示 |
| 主页面 | 气泡系统 | 最佳环境下植物顶部生成氧气气泡 |
| 主页面 | 植物管理 | 最多10株植物、颜色渐变、枯萎纹理 |

## 3. 核心流程

用户打开页面 → 看到预设的三株植物在悬浮草地上 → 通过滑块调节环境参数 → 观察植物颜色变化（健康绿→黄→棕）→ 参数进入最佳范围时氧气气泡释放 → 可添加/移除植物 → 移动端自动切换布局

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深绿 #0a1a0a 背景 + #00cc66 强调色
- 辅助色：健康绿 #4CAF50、黄色 #FFC107、棕色 #795548
- 面板效果：半透明磨砂玻璃 backdrop-filter: blur(10px)
- 字体：无衬线字体，文字颜色 #e0ffe0
- 滑块样式：宽200px高40px圆角矩形，轨道#2c2c2c，滑块头直径20px
- 动画过渡：所有交互0.15秒平滑过渡

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 3D 场景 | 圆形草地(半径20)、星空背景、植物模型(高度2-4) |
| 主页面 | 右侧面板 | 三个垂直滑块、参数标签、当前值显示 |
| 主页面 | 顶部状态栏 | FPS 显示、提示文字 |
| 主页面 | 气泡特效 | 半透明淡绿色球体、正弦波动飘移 |

### 4.3 响应式设计
- 桌面端：右侧垂直参数面板（宽度200px）
- 移动端（<768px）：面板折叠到底部，滑块宽度自适应100%
- 触控优化：支持触屏拖拽旋转、双指缩放

### 4.4 3D 场景指引
- 环境：深黑色星空背景 + 悬浮圆形草地，噪点纹理随光照动态变化
- 光照：环境光 + 方向光，强度随光照参数动态调节
- 相机：透视相机，视角限制在半径40单位球体内
- 植物模型：
  - 蕨类：多层羽状叶片结构
  - 向日葵：圆盘花心 + 黄色花瓣环绕
  - 仙人掌：圆柱体 + 凹陷纹理
- 后处理：无额外后处理，保证50fps以上性能
- 性能预算：最多10株植物、气泡对象池回收
