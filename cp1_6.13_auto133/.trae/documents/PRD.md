## 1. 产品概述

"墨韵·演武"是一款浏览器端的交互式水墨风即时策略小游戏。玩家扮演水墨画师，通过鼠标在宣纸背景上绘制墨水士兵，并指挥它们自动对战，最终击败敌方墨水将军。

- 核心玩法：绘制 + 策略指挥 + 自动战斗
- 目标用户：喜欢休闲策略与水墨美学的玩家
- 技术栈：TypeScript + Vite + Canvas2D + GSAP，无框架依赖

## 2. 核心功能

### 2.1 功能模块

1. **战斗画布页面**：宣纸纹理背景，左右分区（玩家/敌方），绘制士兵、选框指挥、自动对战

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 战斗画布 | 绘制士兵 | 玩家在左侧区域拖拽鼠标绘制笔触，笔触轨迹生成墨水士兵，轨迹越长士兵越大生命越高(50-150) |
| 战斗画布 | 选中与指挥 | 点击选中士兵（金色光环脉动1.5s），再点右侧令其移动(150px/s)并自动攻击(间隔1s,伤害20) |
| 战斗画布 | 框选指挥 | 在己方区域拖拽矩形框选多个士兵（浅墨色选框+深墨色虚线6px/4px），右键点击敌方区域集体移动攻击 |
| 战斗画布 | 敌方AI | 右侧每2-4秒生成深灰墨水士兵，生命和攻击力按时间递增(每分钟生命+10%,攻击+5%)，深灰同心圆纹理 |
| 战斗画布 | 战斗特效 | 攻击时水墨拖尾线条(3px,透明度0.6→0,持续0.3s)；士兵死亡消散动画(5-8墨点+墨色波纹) |
| 战斗画布 | HUD界面 | 左上角玩家存活数、右上角敌方存活数+游戏时间、右下角墨色印章、中央竖线分界 |
| 战斗画布 | 宣纸纹理 | Canvas生成1-2px纤维噪点(透明度0.1)，浅米色背景，边缘毛边效果 |

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：浅米色(#F5F0E8)宣纸背景 + 墨色(#2C2C2C)UI元素
- 辅助色：淡金色(#C4A35A)选中光环
- 士兵风格：半透明墨水团，带墨晕扩散效果；敌方为深灰同心圆
- 字体：水墨数字风格，24px墨色
- 布局：全屏Canvas，左右各占50%，中央墨色竖线分界

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 战斗画布 | 宣纸背景 | 浅米色#F5F0E8，1-2px纤维噪点透明度0.1，边缘毛边效果 |
| 战斗画布 | 分界线 | 中央竖线，墨色2px透明度0.3 |
| 战斗画布 | 玩家HUD | 左上角存活士兵数，水墨数字24px墨色 |
| 战斗画布 | 敌方HUD | 右上角存活士兵数+游戏时间(分:秒) |
| 战斗画布 | 印章 | 右下角方形墨色印章，白色细线小字"墨韵·演武 1.0"，悬停下沉3px加深 |
| 战斗画布 | 选中光环 | 淡金色半透明，直径大30%，脉动周期1.5s |
| 战斗画布 | 框选指示 | 浅墨色半透明填充，深墨色虚线边框(6px/4px) |
| 战斗画布 | 攻击拖尾 | 水墨线条3px，透明度0.6→0，持续0.3s |
| 战斗画布 | 死亡消散 | 5-8墨点(6-15px)+墨色波纹(5→30px) |

### 4.3 性能约束

- 游戏循环60FPS
- 士兵上限30（玩家15+敌方15）
- 粒子总数上限60个
- 粒子生命周期结束立即移除
