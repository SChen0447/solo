## 1. 产品概述

虚拟古法香水调配与嗅觉记忆图谱——让用户化身隐居深山的调香师，在浏览器中从蒸馏器挑选精油、混合香基、记录配方，并生成可交互的嗅觉记忆花瓣图谱。

- 目标用户：对香氛文化感兴趣的创意人群，追求沉浸式交互体验
- 核心价值：以东方古法调香为叙事背景，融合视觉美学与嗅觉隐喻，打造「调香即修心」的沉浸式创作工具

## 2. 核心功能

### 2.1 功能模块

1. **调香室页面**：石砌调香室场景、蒸馏器交互、精油架拖拽、调香碗混合、配方命名保存
2. **嗅觉图谱页面**（集成在同一页面右侧面板）：花瓣图展示已保存配方、悬停详情浮窗、点击重新加载

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 调香室 | 蒸馏器 | 中央橡木桌上摆放铜盖玻璃蒸馏器（SVG），点击可从精油架获取香气；加热时底部火焰粒子动画（6个橙色小圆点随机缩放，透明度0.6→0.3循环），瓶内液体冒泡 |
| 调香室 | 精油架 | 左侧展示6种精油（薰衣草/柠檬/檀木/茉莉/薄荷/玫瑰），水晶瓶形图标+横条文字标签；悬停时瓶身浮现对应微光光晕并显示香调描述 |
| 调香室 | 调香碗 | 碗口直径140px，碗沿0.5px铜色描边；拖拽精油入碗后液体颜色按比例混合叠加，碗底浓度进度条（15px高度，颜色随混合液渐变）；4种以上精油时碗周围飘出5-8个半透明香气粒子 |
| 调香室 | 配方保存 | 输入配方名和简短评语（最多80字），保存到图谱（最多12条） |
| 图谱面板 | 花瓣图 | 每条配方对应一片花瓣，颜色取配方主色，花蕊显示配方名；悬停弹出浮窗展示精油比例柱状图+评语；点击花瓣重新加载配方到调香碗 |

## 3. 核心流程

用户进入调香室 → 浏览精油架 → 拖拽精油到调香碗 → 观察混合颜色变化 → 4种以上时感受香气粒子飘散 → 命名并保存配方 → 右侧图谱面板生成花瓣 → 悬停查看配方详情 → 点击花瓣重新加载调整

## 4. 用户界面设计

### 4.1 设计风格

- **主色系**：温润羊皮纸色系，主背景 #f5ede0，辅色 #7a6952
- **渐变背景**：暖石灰 #d1c6b5 → 旧木褐色 #8b735a
- **交互按钮**：铜色填充 #b89a6e，悬停变 #c8aa7e（0.2秒 ease-out 过渡）
- **圆角**：8px，微弱盒阴影（偏移2px，扩散6px，透明0.15）
- **拖拽交互**：拖拽中缩放1.05倍且半透明
- **字体**：选择具有古典手写感的衬线字体（如 Noto Serif SC），搭配等宽标签字体

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 调香室 | 场景背景 | 石砌墙壁纹理，顶部透光效果，暖石灰到旧木褐渐变 |
| 调香室 | 橡木长桌 | 中央桌面元素，放置蒸馏器和调香碗 |
| 调香室 | 蒸馏器 | SVG铜盖玻璃瓶，底部火焰粒子动画 |
| 调香室 | 精油架 | 左侧6个水晶瓶+标签，悬停微光光晕 |
| 调香室 | 调香碗 | 140px碗口，铜色描边，混合液颜色叠加，底部浓度进度条 |
| 调香室 | 香气粒子 | 5-8个半透明圆，6-12px直径，缓慢上升飘散 |
| 图谱面板 | 花瓣图 | 圆形排列花瓣，花蕊配方名，悬停浮窗含柱状图+评语 |

### 4.3 响应式适配

- **桌面端**：主调香区占60%，图谱面板占35%，左右布局
- **平板端**：调香区和图谱面板上下排列
- 触摸优化：拖拽操作兼容触摸事件

### 4.4 性能要求

- 交互响应延迟 < 100ms
- 动画帧率 ≥ 30fps
- 香气粒子使用 requestAnimationFrame 优化
