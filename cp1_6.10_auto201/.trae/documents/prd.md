## 1. 产品概述

「共读圈」是一个面向独立书店读者的在线共读活动平台，让读者可以发起共读计划、记录阅读进度、撰写批注笔记，并与其他共读者实时交流心得。

- 目标用户：独立书店的读者群体，热爱阅读并希望与他人分享阅读体验
- 核心价值：构建虚拟阅读社交空间，解决独自阅读缺乏交流的痛点
- 市场定位：纯前端轻量级应用，不依赖后端服务，适合快速部署和演示

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 读者用户 | 本地模拟登录 | 发起/参与共读、记录进度、撰写批注、实时聊天 |

### 2.2 功能模块
1. **共读列表页**：卡片网格展示所有共读计划，支持点击进入详情
2. **共读详情页**：书籍信息展示、阅读进度、批注笔记区、实时聊天区
3. **个人阅读档案页**：参与历史、阅读统计、批注数量展示

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 共读列表页 | 卡片网格 | 默认4列/移动端2列响应式布局 |
| 共读列表页 | 共读卡片 | 封面、书名、日期、成员数、进度条，悬停动效 |
| 共读详情页 | 书籍信息卡 | 左侧40%，封面、书名、作者、进度条 |
| 共读详情页 | 批注笔记区 | 中间35%，毛玻璃笔记卡片、时间倒序、滑入动画 |
| 共读详情页 | 实时聊天区 | 右侧25%，滚动消息、发送/清空、发送提示 |
| 个人档案页 | 数据卡片 | 三张渐变卡片、弹性弹出动画、数字计数器 |
| 全局 | 导航栏 | 固定顶部、毛玻璃效果、页面高亮、汉堡菜单 |

## 3. 核心流程

用户进入共读圈首页，浏览所有共读计划卡片，点击感兴趣的共读进入详情页。在详情页可查看书籍信息和整体进度，阅读其他成员的批注笔记，也可发布自己的批注。同时可在右侧聊天窗口与其他正在阅读的成员实时交流。用户可通过导航栏进入个人档案页，查看自己的共读历史和阅读统计数据。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：#121212（深色背景）、#1e1e2e（卡片基底）
- 强调色：#bb86fc（紫罗兰）、#03dac6（青绿）、#cf6679（粉红警示）
- 按钮风格：圆角12px，悬停缩放1.05倍，颜色过渡0.2秒
- 字体：全局字体优先，标题加粗，正文适中
- 布局：居中容器（max-width: 1200px），卡片圆角12px，8px内阴影
- 图标风格：简洁线性图标，配合强调色

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 共读列表页 | 卡片网格 | 4列网格，768px以下2列，悬停上浮6px+渐变描边，进度条0%→当前值动画1.2秒 |
| 共读详情页 | 批注笔记区 | 毛玻璃效果（rgba(255,255,255,0.12)+8px模糊），悬停展开+高亮用户名，新笔记滑入0.5秒 |
| 共读详情页 | 实时聊天区 | 固定高度90%，自动滚动到底部，发送按钮#6200ee→#3700b3，消息提示2秒消失 |
| 个人档案页 | 数据卡片 | 三种渐变色背景，从底部弹出间隔0.2秒，数字计数器1.5秒ease-out |
| 全局 | 导航栏 | 固定顶部，#121212+10px模糊毛玻璃，当前页#bb86fc高亮，头像粉色#ff4080发光 |

### 4.3 响应式设计
- 桌面端优先，最大宽度1200px居中
- 768px以下：共读列表变2列，详情页三栏变单列
- 600px以下：单列布局，所有动画关闭，导航栏折叠为汉堡菜单（左侧滑出，半透明模糊背景）

## 5. 性能要求
- 页面切换：≤200ms（React Router懒加载）
- 列表渲染：批注和聊天列表≤50ms（虚拟列表概念模拟）
- 动画帧率：计数器动画期间≥55fps
- 运行方式：npm install && npm run dev
