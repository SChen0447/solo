## 1. 产品概述

音乐主页生成器是一款面向独立音乐人的轻量级可视化搭建工具，用户无需编程技能，通过拖拽组件和配置参数即可快速生成个性化音乐宣传主页。

- **核心价值**：降低独立音乐人展示作品的技术门槛，提供专业、美观、响应式的个人音乐主页
- **目标用户**：独立音乐人、乐队、音乐创作者、播客主播
- **解决问题**：无需代码知识，通过拖拽式操作快速生成具备播放器、歌单、社交链接的宣传页面

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 独立音乐人 | 无需注册，直接使用 | 拖拽组件、编辑属性、导出配置、复制嵌入代码 |

### 2.2 功能模块

1. **组件工具栏**：音乐播放器、歌单列表、社交链接三种可拖拽组件
2. **画布编辑区**：组件放置、拖拽排序、位置调整
3. **属性配置面板**：选中组件的详细属性编辑
4. **主题系统**：三种预设主题 + 自定义颜色选择器
5. **导出功能**：JSON配置导出 + HTML嵌入代码复制

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑器 | 左侧工具栏 | 展示三种可拖拽组件类型，支持拖拽到画布 |
| 主编辑器 | 中间画布 | 渲染已放置组件，支持拖拽排序、双击编辑 |
| 主编辑器 | 右侧属性面板 | 根据选中组件显示对应编辑表单，实时预览 |
| 主编辑器 | 顶部主题栏 | 主题切换、自定义颜色、导出功能 |
| 模态编辑器 | 播放器编辑 | 封面图片、背景渐变色、播放模式设置 |
| 模态编辑器 | 歌单编辑 | 歌曲增删、封面设置、标签颜色自定义 |
| 模态编辑器 | 社交链接编辑 | 链接增删、图标样式选择（最多5个） |

## 3. 核心流程

用户从左侧工具栏拖拽组件到中间画布，通过双击或右侧面板编辑组件属性，切换主题预览效果，最后导出配置或复制嵌入代码。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色主题（默认背景 #1a1a2e），辅以霓虹渐变点缀
- **视觉风格**：磨砂玻璃效果（backdrop-filter: blur 12px），卡片式布局
- **按钮风格**：圆角按钮，hover 时上移 2px 并加深阴影，过渡动画 0.3s
- **字体**：现代无衬线字体，标题加粗，正文清晰易读
- **动效**：拖拽弹性回弹、主题切换平滑过渡（0.6s）、输入框聚焦发光

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主编辑器 | 工具栏 | 组件卡片、拖拽图标、半透明阴影、回弹动画 |
| 主编辑器 | 画布 | 浅灰网格背景、组件卡片、选中高亮、拖拽避让 |
| 主编辑器 | 属性面板 | 表单输入、颜色选择器、实时预览 |
| 主编辑器 | 顶部栏 | 主题切换按钮、颜色选择器、导出按钮 |
| 模态框 | 编辑弹窗 | 毛玻璃背景、下划线发光输入、保存按钮 |

### 4.3 响应式

- **桌面端**（≥1024px）：三栏布局 - 左工具栏280px / 中画布自适应 / 右属性面板300px
- **平板/移动端**（<1024px）：工具栏变为顶部横向滚动，属性面板收起为底部抽屉浮层
- **触控优化**：增大点击区域，支持触摸拖拽

### 4.4 动画与交互

- 组件拖拽：跟随鼠标 + 半透明阴影 + 弹性回弹
- 组件避让：拖拽时其他组件弹性让位动画
- 主题切换：所有颜色、字体、阴影平滑过渡 0.6s
- 输入聚焦：下划线发光动画
- Toast 提示：右下角绿色提示，3秒后淡出
- 按钮 hover：上移 2px + 阴影加深
