## 1. 产品概述
虚拟魔法植物培育园 - 一个沉浸式3D互动Web应用
- 用户扮演植物炼金术师，在悬浮星尘土壤上通过混合元素液滴培育独一无二的发光魔法植物，并在公共温室展示
- 核心价值：创造性的元素配比系统、视觉震撼的3D植物生长动画、社区互动点赞排行榜

## 2. 核心功能

### 2.1 功能模块
1. **首页培育台**：3D星尘培育场景、元素水晶交互、种子培育、植物生长
2. **植物详情面板**：元素配比饼图、生长天数、点赞数展示
3. **公共温室**：排行榜展示、3D缩略模型展示、点赞交互

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页培育台 | 3D星空场景 | 深蓝到紫罗兰渐变背景，悬浮椭球培育台，400个星点粒子，4个元素水晶 |
| 首页培育台 | 元素水晶交互 | 点击火/水/风/土水晶，液滴沿抛物线飞向种子，带拖尾粒子 |
| 首页培育台 | 种子培育系统 | 最多3颗种子，每颗5滴元素液滴，颜色扩散动画，萌芽生长 |
| 首页培育台 | 植物生长系统 | 5秒生长动画，元素配比决定形态，25-40个光点粒子环绕 |
| 植物详情 | 配比信息 | 元素配比饼图展示 |
| 植物详情 | 操作按钮 | 点赞、查看详情 |
| 公共温室 | 排行榜 | 最近24小时点赞Top10植物，每行3株，网格布局 |
| 公共温室 | 植物卡片 | 3D缩略模型旋转，底部滑入动画，点赞星星动画 |

## 3. 核心流程
用户进入首页 → 查看3D星空培育场景 → 点击元素水晶选择元素液滴 → 液滴飞向种子 → 种子颜色变化 → 满5滴后萌芽生长 → 植物成熟 → 点击植物查看详情 → 进入公共温室查看排行榜 → 为喜欢的植物点赞

## 4. 用户界面设计
### 4.1 设计风格
- 主背景：#05081a（深蓝）#2a0a3a（紫罗兰）
- 点缀色：#ffd700（金色），#00ffcc（亮青）
- 元素色：火#ff3300、水#0066ff、风#33cc33、土#cc8833
- 按钮风格：圆角卡片式，悬停放大1.08倍+0.3s光晕过渡
- 字体：使用具有未来感的无衬线字体
- 布局：全屏沉浸式3D画布，悬浮UI面板
- 图标风格：发光霓虹风格，半透明玻璃质感

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页培育台 | 3D场景 | 全屏Three.js画布，渐变背景，悬浮培育台居中 |
| 首页培育台 | 元素水晶栏 | 底部横向排列4个发光水晶，悬停放大 |
| 首页培育台 | 导航按钮 | 右上角公共温室入口，金色发光按钮 |
| 植物详情 | 详情面板 | 右侧滑出面板，玻璃质感，配比饼图居中 |
| 公共温室 | 温室页面 | 彩虹渐变玻璃背景，3列网格布局 |
| 公共温室 | 植物卡片 | 3D缩略图，底部信息，点赞按钮 |

### 4.3 响应式
- Desktop-first设计
- 宽度<768px时：公共温室每行1株，隐藏部分粒子特效提升帧率

### 4.4 3D场景指导
- 环境：深蓝#05081a到#2a0a3a径向渐变星空，雾气效果
- 光照：环境光+方向光+点光源，突出发光效果
- 相机：透视相机，视角60度，距离8单位
- 构图：培育台居中，元素水晶环绕分布
- 交互：Raycaster点击检测，平滑相机动画
- 后处理：Bloom发光效果，抗锯齿
- 性能预算：目标60fps，粒子数优化，移动端降质
