## 1. 产品概述

多轨音频混音器 Web 应用，面向独立音乐人，提供浏览器内多轨乐器演奏音频的混合、实时预览与导出下载。

- 解决音乐人无需专业 DAW 软件，通过浏览器即可完成多轨混音工作流
- 目标用户：独立音乐人、Podcast 制作人、音乐爱好者

## 2. 核心功能

### 2.1 功能模块
1. **主混音面板**：轨道列表展示、音量/平衡/静音/独奏控制、波形缩略图
2. **文件上传模块**：多音频文件批量上传（最多 4 轨，支持 MP3/WAV）
3. **播放控制模块**：播放/暂停、进度指示、进度跳转
4. **导出模块**：混音合成、WAV 文件下载、加载动画

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|----------|----------|
| 混音工作台 | 文件上传区 | 拖拽或点击上传最多 4 个音频文件，显示轨道 |
| 混音工作台 | 轨道卡片 | 每个轨道展示名称、波形缩略图、音量滑块、平衡旋钮、静音/独奏按钮 |
| 混音工作台 | 全局播放控制 | 播放/暂停按钮、进度指示、可拖动进度条 |
| 混音工作台 | 导出区域 | 导出混音按钮、全屏加载遮罩 |

## 3. 核心流程

用户上传多个音频文件 → 每个文件解析为独立轨道 → 用户实时调整各轨道音量与平衡参数 → 实时预览混音效果 → 用户点击导出 → 浏览器内合成 WAV → 下载到本地

```mermaid
flowchart LR
    A["用户上传音频文件"] --> B["解析为 AudioBuffer"]
    B --> C["渲染轨道卡片与波形"]
    C --> D["用户调整音量/平衡/静音/独奏"]
    D --> E["Web Audio API 实时混音预览"]
    E --> F["点击导出混音"]
    F --> G["离线合成 PCM 数据"]
    G --> H["生成 WAV Blob 并触发下载"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 深色主题：背景 `#1a1a2e`，面板 `#16213e`
- 毛玻璃效果 `backdrop-filter: blur(6px)`
- 轨道主题色：钢琴 `#ff6b6b`、吉他 `#4ecdc4`、鼓 `#ffe66d`、其他 `#95e1d3`
- 圆角矩形轨道卡片 `border-radius: 12px`
- 按钮/滑块弹性动画 `transition: all 0.2s`
- 交互反馈：轨道卡片微缩放 `transform: scale(0.98)`

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 混音工作台 | 文件上传区 | 拖拽区域、文件选择按钮、文件数量限制提示 |
| 混音工作台 | 轨道卡片 | 轨道名称、Canvas 波形、音量滑块、平衡旋钮、静音按钮、独奏按钮 |
| 混音工作台 | 播放控制栏 | 播放/暂停按钮、进度指示条（带拖动进度绿线 |
| 混音工作台 | 导出区 | 导出按钮、全屏遮罩、旋转圆环加载动画 |

### 4.3 响应式设计
- 桌面端（>=768px）：四列轨道排列
- 平板（>=768px 单列或四列，手机（<768px）单列
- 整体采用 Flex 居中布局

### 4.4 性能约束
- 混音预览与导出操作须在用户操作后 300ms 内开始渲染
- 4 轨每轨 3 分钟混音导出不超过 2 秒
