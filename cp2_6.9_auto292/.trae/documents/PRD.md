## 1. 产品概述

"时光配方"是一款让用户通过选择和组合不同风格的"气味音符"，为回忆时刻生成定制虚拟香水瓶卡片的Web应用。用户可以像调制香水一样创作专属配方，生成带有水彩风格瓶身插图的卡片，并与他人分享、品鉴和收藏。

- 核心价值：将抽象的回忆时刻转化为可感知的嗅觉艺术作品，提供创造性的情感表达体验
- 目标用户：喜欢香水、艺术创作和记录生活的年轻用户群体

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，输入昵称即可使用 | 创建配方、浏览卡片、品鉴评分、收藏配方 |

### 2.2 功能模块

1. **首页/配方创建页**：气味音符选择面板、浓度比例滑块、回忆文本输入、调香师签名、调香按钮
2. **卡片画廊页**：无限滚动卡片网格、搜索过滤、排行榜展示
3. **卡片详情页**：完整配方展示、关键词灵感来源、品鉴评分、收藏功能

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 气味选择面板 | 4组气味类别（柑橘、花香、木质、皮革），每组5个音符，可选1主调2辅调 |
| 首页 | 浓度调节 | 滑块控制每个音符浓度（10%-100%），圆环图可视化展示 |
| 首页 | 回忆输入 | 1-2句场景描述（最多100字），自动提取3个关键词 |
| 首页 | 调香按钮 | 提交配方生成卡片，调用后端API |
| 画廊页 | 卡片网格 | 3列瀑布流布局，卡片悬停上浮动画，无限滚动加载 |
| 画廊页 | 搜索框 | 300ms防抖搜索，匹配名称和关键词，匹配文字高亮 |
| 画廊页 | 排行榜 | 按品鉴数降序排列，实时刷新 |
| 详情页 | 卡片展示 | 完整SVG香水瓶插图，包含瓶身、水彩纹理、用户缩写 |
| 详情页 | 品鉴评分 | 5星评分组件，悬停变色，提交后品鉴数+1 |
| 详情页 | 配方说明 | 完整配方列表、浓度比例、关键词灵感来源 |

## 3. 核心流程

用户进入首页后，从4组气味类别中各选择1个主调和2个辅调，通过滑块调整浓度比例，输入回忆场景描述和调香师签名，点击"调香"按钮。系统将配方发送至后端，生成随机水彩风格SVG香水瓶卡片，保存至内存数据库。生成的卡片自动出现在画廊中，用户可滚动浏览、搜索过滤、点击查看详情、进行品鉴评分。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：暖色系，主色#F5E6C8、辅色#D4A574、强调色#6C63FF
- **背景**：浅米色到淡橙色渐变（#FFF8F0到#FFE8D0）
- **卡片样式**：圆角12px，阴影0 4px 12px rgba(0,0,0,0.08)，背景色#FFF
- **字体**：标题使用Georgia衬线字体，正文使用Inter无衬线字体
- **动效**：音符选中时translateY(-4px)上浮，卡片悬停时translateY(-6px)并增加阴影，过渡0.2-0.3s

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|----------|
| 首页 | 气味选择面板 | 圆角卡片布局，圆形音符图标，选中时1px实线边框#6C63FF，上浮动画 |
| 首页 | 浓度滑块 | 轨道高4px圆角2px背景#E0D0B8，滑块头直径20px颜色#6C63FF，拖拽时放大到24px并发光 |
| 首页 | 圆环图 | 可视化展示各音符浓度比例 |
| 画廊页 | 卡片网格 | 每行3列，间距16px，瀑布流布局，悬停上浮6px阴影加深 |
| 详情页 | 评分组件 | 5个星形图标，悬停变色#FFD700 |
| 全局 | 搜索框 | 顶部搜索栏，300ms防抖，匹配文字高亮#6C63FF |

### 4.3 响应式设计

- 桌面优先设计，1024px以上分辨率正常布局
- 768px以下切换为单列布局
- 触摸设备优化交互体验

### 4.4 性能要求

- 页面首次加载时间≤3秒（代码分割+懒加载）
- 搜索响应延迟<200ms（包含300ms防抖）
- 卡片列表滚动帧率≥50fps（虚拟滚动技术）
